我只是坐在那里打游戏，就推动了科学发展。。。

昨天，差评君正在和小黑胖挑灯夜战，企图补上那个发售已久但我们却一直没空打完的游戏《 无主之地 3 》的最新 DLC ——《 枪炮、爱情与触手之婚礼大作战》。

在主角一行人的大本营——宇宙飞船避难所 3 号上，我们在坦尼斯的实验室里发现了一台游戏机。

当然，之后我们发现这玩意儿 4 月份的时候就躺在那儿了，这份惊奇只是因为我俩 AFK 这个游戏太久了导致的。。。

扯远了。

总而言之，当时我俩还是充满了好奇的，兴冲冲地就跑到跟前开启了这个名为 “ 无主之地科学 ” 的街机，成为了 “ 在游戏中玩游戏 ” 的两名套娃。

然后我就惊了，这特喵的原来是个科研项目！

开发团队 Gearbox 还请到了《 生活大爆炸 》里艾米的演员 Mayim Bialik 给我介绍起了这个项目，熟悉她的差友们应该都知道，她是个博士。

大致意思是现在的科学家试图研究人体内的微生物，从而更好地理解人体而帮助医学发展。

要研究这些微生物，有一个重要步骤是根据它们的 DNA 相近程度来进行分类。

在科研领域，一般这种计算量巨大的活儿都交给电脑干。

计算机擅长与数字打交道，做起 DNA 简单分类的工作却比较吃瘪，还老犯错。

但人类很擅长做这类简单的分类工作。

于是做这个项目的 Microsetta Initiative 就联系上了 Gearbox ，在它们最受欢迎的游戏《 无主之地 3 》里添置了一个迷你游戏。

游戏有 4 种不同颜色的砖块，每一行要求出现某种砖块，对上以后就能得分。

而玩家只要在有限的次数内顶起某一列的砖块，尽可能顶到一定数量就可以过关了。

游戏一共有无限关，不同色彩的砖块代表着不同的核苷酸，玩家试图排列砖块的过程也就是在给 DNA 排列到正确的分类里。

哪怕排出来的结果并不完美，只要玩家尽量尝试了，也能帮助科学家研究出更好的算法。

为了激励人们去玩，这个小游戏还设置了奖励，每过关到一定阶段可以获得一些新奇的人物道具。

或者是拿着奖币换一些强化游戏奖励的消耗品。

总之，为了人类科学发展（ 其实是为了游戏奖励 ），我和小黑胖玩了这个小游戏很久，结果搁置的任务一晚上过去了依旧没什么进展。。。

其实这并不是第一次游戏界和科学界搞这种梦幻联动，网游 EVE 在 2015 年也做了一个，他们的项目叫做 Project Discover 。

在游戏里，玩家可以帮助给蛋白质图像做分类工作，这个科研项目叫做「 人类蛋白质图集 」。

后来 Project Discover 在一两年后又整了个新姿势，叫做 Exoplanets ：让玩家通过给天文望远镜拍摄到的光谱做标记，帮助科学家和科研计算机寻找地外行星。

这俩项目看起来挺费脑子，其实只是简单的分类工作，相信我如果你玩过 EVE ，就会知道和这个极其复杂的游戏比，这俩科研小游戏算不上啥。。。

其实这些大游戏里的科研小游戏的背后，是一门新兴领域，叫做 Citizen Science ，翻译过来叫做「 公民科学 」。

「 公民科学 」由科学家们主导科学研究，普通人参与一些基本的数据收集整理分类等高重复低门槛的工作。

同样是普通人参与的科研工作，「 公民科学 」和那些在家里研究永动机的「 民科 」不是一回事儿。。。

差评君以前介绍过 BONIC ：一个闲时贡献自己电脑算力给科研项目的平台，至今我电脑上还跑着它。

「无主之地科学」和 EVE 的「 Project Discovery 」需要解决的问题不一样，它们缺的不是算力，而是人。

电脑做数据计算飞快，但**类似给图片分类打标签的事儿其实能力有限——这些活儿在生物科学、天文学等领域非常多****，**可电脑就是做得不够麻利还容易犯错，这些没营养的琐事让科学家们也很蛋疼。

以往这种事儿不得不甩给刚进实验室的研究生，和刚进厨房先学淘米洗菜差不多意思，后来我们有了发达的互联网和分布式计算。

但话又说回来了，这些枯燥简单的工作对科研工作者没意思，对普通人怎么就会有意思呢？普通人确实可以帮上忙，但凭什么帮忙呢？

科学家们这么聪明怎么会想不到这点呢？

于是他们找到了游戏开发者，让它们变成了游戏的样子。。。

最初，他们怀着这样的想法专门开发了相应的游戏。

2008 年一款叫做 Foldit 的网络游戏发布了，它要求玩家用游戏提供的工具把一些蛋白质尽可能折叠成一个完美的形状，科学家们则可以拿结果来研究蛋白质，推动生物和医疗科学的发展。

2012 年，又有一款叫 EyeWire 的游戏发布，主攻神经科学。

这俩先驱游戏其实都是很好的项目，但差评君挨个儿玩了以后不得不说：它们不是很好玩。。。一个游戏得好玩儿啊！

当时应该也有人和差评君想到一块儿去了，于是换了个思路，先去找本身就很好玩的游戏，再想办法和科研项目结合。

于是就有了 EVE 和《 无主之地 3 》里的小游戏**，还用奖励诱惑你。。。**

在 EVE 和《 无主之地 3 》科研项目的背后，是一个叫做 MMO 的平台，专门帮助科学家把这类问题转化成游戏问题，他们是这么说的：

「 玩家整天在虚拟世界里解决各种难题，他们完全有能力为这类基础的现实问题做出贡献 」

「 人们都爱玩游戏。据统计全人类花在《 魔兽世界 》上的时间大约是六百万年，想想看植入小游戏科研从中转化哪怕一点点时间会怎么样？ 」

说实话，差评君觉得他们讲得挺有道理的。。。而且身为一个玩家我也丝毫不反感做这些事儿。

我愿意为科学点亮我的显卡。

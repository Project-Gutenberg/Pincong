我的手机里插着一张香港的SIM卡（prepaid），但一般不启用。今天因为要收短信就启用了它。该SIM卡在内地漫游对应的运营商是中国联通，结果发现手机里进来一条短信：

Welcome to China. Enjoy the best network coverage and super-fast surfing only on China Unicom’s mobile broadband network. Please use + or 00 before the country code for international call. Have a pleasant stay.

我在想：明明它可以写成 Welcome to Mainland China，结果它直接写成了 China，这不明摆着告诉大家香港不属于 China 吗？

这条短信实际上是所有国际漫游的模版，小粉红们对辱华行为平时各种鸡蛋里挑骨头无孔不入，联通在后台增加一个 if else 判断一下 SIM 卡来源地然后修改一下模版也就一分钟的事情，是不是明知有这样的bug故意放着不管的？

五月的第二个星期天是**“母亲节”，**就在全网热搜“母亲节送什么给妈妈”的时候，另一条关于母亲的新闻也让全网震惊。

陕西一男子将79岁近乎瘫痪的母亲扔进废弃墓坑并将其活埋。

现在只要在微博输入**“活埋”二字，就会看到多条相关新闻被顶上了热搜，就连国家卫健委和全国老龄办**都发声了，可见此事带来的社会影响有多恶劣。

中国自古以来就是一个讲**“孝道”的国家，孝顺父母，赡养老人是子女应尽的义务，这种令人发指的事情已超乎了人们的想象，突破了做人的底线。**

到底是什么催生了这种惨剧的发生？

01

5月2日晚，陕西靖边一名58岁的男子马某宽用手推车将79岁的母亲王某芳推出家门，并活埋于离家大约三公里的一处废弃墓穴内，然后于次日凌晨回到家中。

他告诉妻子张某，自己已经雇了一辆面包车将母亲送回到甘肃亲戚家里，张某前往车站并没有见到婆婆的踪影，回到家后发现丈夫也不见了。

焦急的妻子一直苦等丈夫和婆婆回家，可她等到5月3日晚11时左右，仍未等到他们回来，情急之下，张某选择了报警。

靖边警方在接到报案后，立即对马某宽和王某芳的行踪展开了调查，终于在5月5日找到了马某宽。

他一直坚称母亲此时已在甘肃亲戚家里了，但接受民警问询时，他面露慌张，言辞闪烁。

民警在和甘肃亲戚方核实后，确定了马某宽在说谎，于是对其加大了审查的力度，马某宽最终交代了母亲的下落，她被活埋在万亩林一处废弃墓穴里。

5月5日下午4时左右，一众办案民警还有马某宽，在法医和技术人员的陪同下来到了事发地点，经过了**一个多小时的挖掘，**民警听到墓坑里还有动静。

就这样，在被活埋三日后，老人终于**被警方成功救出，**并被送往了当地医院进行救治。马某宽看到母亲还活着，面无表情一言不发。

5月8日，靖边警方表示，马某宽没有精神疾病史，活埋母亲的举动蓄谋已久，现以将其涉嫌故意杀人罪进行批捕并刑拘。

目前王某芳的生命体征也已平稳，可以进行沟通交流。

02

随着事件关注度的不断升高，越来越多的细节被不断披露。

马某宽的**父亲在他很小的时候就去世了，母亲王某芳在他12岁的时候带着弟弟妹妹改嫁甘肃，**将马某宽一人留在了陕西，由叔父进行照顾。

他和叔父并**没有任何血缘关系，**自己的父亲其实是被抱养回家中的。

后来母亲与第二任丈夫生下一个儿子，在第二任丈夫去世后，母亲带着弟弟回到了陕西，此时马某宽已**40多岁了，**后来母亲一直都跟弟弟住在一起。

去年下半年，年事已高的母亲腿脚不便，失去了**自理能力，**就被送到了大儿子家中由他照顾。

有一次母亲不慎在家中摔倒，儿子和儿媳都不在家。摔伤后王某芳就一直**卧床不起，**饮食起居包括上厕所都是儿子和儿媳负责。

他们不在家的时候，母亲也会在床上大小便，刺鼻的恶臭让马某宽难以忍受。

有消息表示，此前一直是弟弟在照顾母亲，后来他生病了母亲才回到大儿子那里。

然而事情并非如此，弟弟是低保户，他身材矮小还伴有严重的驼背，因此至今也没有结婚，母亲带他回到陕西后，一直和他住在一起便于照顾。

马某宽小时候就被母亲遗弃，再次见到母亲后也没能得到母亲的关注，母亲需要养老了才想到自己，这些怨气让他最终对母亲痛下杀手。

马某宽的作案动机让舆论的风向产生了一些变化，有少数网友想为大儿子洗白。

他们表示母亲改嫁，偏心弟弟，马某宽从小就缺爱，而且**“久病床前无孝子”，这都是母亲种下的果，都是原生家庭**害的。

可这些是为马某宽开脱罪名的理由吗？不管大儿子有千千万万个委屈和怨恨，但都**不能够去杀人，**而且还是杀害自己的母亲！

是不是所有原生家庭有问题的人都可以把父母给活埋了？

这个世界上有很多人日子比马某宽苦得多，可他们一生都与人为善；也不乏父母卧病在床几十年，仍将其照顾得妥妥当当的孝子孝女们存在。

任何事情都不能成为你犯罪的理由，人生虽苦，但请你务必善良。

农村妇女王某芳，在丈夫离世后，只能指望着儿子给自己养老了。马某宽是极端的个例，但在农村还有千千万万个像王某芳一样需要被赡养的老人。

农村的养老问题在很多层面上依靠的都是**道德和情分的维系，**一旦失去了，矛盾也就不断被激化，家人的白眼、嫌弃、孤独感和无力感让很多老人最后选择自杀，不得善终。

随着中国老龄化现象的不断加重，中国农村老人的自杀现象近些年来有大幅度上升的趋势，这是我们社会一道不可言说的“伤疤”。

研究显示，在农村，每10万65岁以上的老人中就有21.99人自杀，而且这一数字还随着年龄的增加而增加。

对于85岁以上的农村老人，同比数字则上升到了65.6人；****上吊、喝农药和投水是他们自杀最常见的方式。

农村男性老人自杀率远远高于女性。

以上这些都还只是2018年的数据。

农村养老制度的缺失也是催化王某芳悲剧产生的重要因素，我国农村社会现正处在转型中，如何让老人**“老有所乐，老有所养”，**还有一条漫长的道路要走。

03

对于那些想让父母安享晚年的子女，但苦于自己没有能力对父母进行妥善地赡养，他们该怎么办？

社会是不是也该进行反思，**养老体系如果能够进一步得到完善，**我们是不是也就不会再看到这种新闻了呢？

就以养老制度较为完善的英国为例，随着人口的快速老龄化，养老日益成为了英国不可忽视的社会问题。

因此英国的养老制度逐步从以前的以政府为主，渐渐过渡到了现在以政府、企业和个人三方来共同进行。

首先就是政府层面的两类养老金。

第一类：国家基础养老金（The Basic State Pension）

在1951年4月6日之前出生的男性和1953年4月6日之前出生的女性可以领取国家基础养老金。

要获得该养老金，必须已经缴纳过30年的国民保险（National Insurance）。

目前，可以获得的最高金额是**每周134.25英镑。**如果缴纳年限低于30年，则每周金额将低于134.25英镑。

如果自己没有资格获得该养老金，在配偶满足条件的前提下可以通过配偶来获得每周80.45英镑的养老金。

如果是在上述年龄之后出生的公民则需要参考最新的国家基础养老金（The New State Pension）。

最早可获得此养老金的时间是公民达到国家养老年龄（65岁），缴纳至少10年的国民保险（National Insurance）。

第二类：国家补充养老金（Additional State Pension）

1951年4月6日之前出生的男性和**1953年4月6日之前出生的女性，**可以在国家基础养老金之上获得此项额外金额。

需要注意的是，领取过新的国家基础养老金的人则不能再次领取补充养老金，但如果配偶去世，则可以从配偶处那里继承。

除了以上两种养老金，还有企业层面的职业养老金计划（Workplace pension scheme）。

雇主必须对其雇员实行职业养老金计划。如果雇员年龄在**22岁至国家退休年龄之间，年收入至少在10000英镑，在英国境内工作，**那么雇主就会自动为雇员加入职业养老金计划。

养老金由自己工资支付的部分，雇主交付的部分和政府税务减免三部分组成。

英国还存在很多种**个人养老金计划，**就这些个人类别的养老金最好向相关人士进行专门的咨询。

以上几种养老金模式基本上构成了英国养老金的三大来源。

还有发展已较为完善的以社区为依托，政府发挥主导作用的英国“社区养老”模式也是值得我们去借鉴和学习的。

我们梳理事件的前因后果，对比英国养老制度并不是要抨击**“中国式养老”，**而是希望从多个角度对这件事进行探讨，除了当事人的丧尽天良以外，整个社会特别是养老系统和体系中的一些问题，也是需要我们进一步思考和探索改进的地方。

知乎上曾经有人写出自己的家外公住院时隔壁的真实养老情况。隔壁病房的老人是一个退休高干，今年已经90岁，每个月的退休金奖金1.5万RMB，公费医疗。

俗话说“久病床前无孝子”，在这个病房里是不存在的事。全家人都巴不得老人能够寿比南山，哪怕只是无意义地用呼吸机吊着也希望能让他的生命在物理意义上延续下去。

因为只要老人活着，啥也不干，家里就能净入账1.5万RMB；

因为只要老人活着，就可以一直住在医院的单人间病房里，享受着最好的医疗服务；

因为只要老人活着，家属从医院领的药品通通可以报销，全家人笑逐颜开。

但这只是一少部分老人才会拥有的待遇。而贫困的农村老年人才是大多数，他们也是最无助的存在。除去那些像华西村一般自建养老体系的东部富裕农村，绝大多数的农村老人除了规定的每月几十元养老金，再没有任何固定收入。

他们没有办法，只能依靠自己的子女坐吃山空。手停即口停。这对子女的家庭又是一笔负担，尤其是子女也是普通农村人的时候。

最后，引用知乎上一位作者的一句话：

“精准扶贫，绝不是一句口号！是拯救苍生的重要工程”。

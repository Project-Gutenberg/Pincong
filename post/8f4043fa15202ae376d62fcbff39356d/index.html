南方恐怖故事：成群结队的白蚁又来啦

每当夏季雨水来临前后，我国南方局部地区的朋友们总会因一类小虫子而愁眉不展，甚至惊慌失措。

在南方中低海拔地区夏季的黄昏，山区林地附近屋子昏暗的灯光下，很快会出现一只扇着四片狭长灰翅的小虫子，接着是两只、三只、千千万万只……而在中高海拔的西南地区，五六月中午雨霁[jì]时分，这类小虫子会从河岸、堤坝、山林等地，一群群地缓缓飞向天空，它们扇动翅膀的轻柔样子，像极了旋转上飞的灰色蒲公英。

灯下飞舞的白蚁群。图片：TOPick 新聞 / youtube

其实，这类虫子就是恶名远播的白蚁，而且还是白蚁的繁殖蚁。它们在进行大型多对多相亲交流会。

一场大型的不可描述

严格来说，白蚁的婚飞应该被称为“分飞”。分飞指的是，雌雄有翅成虫在环境条件适宜时，从母巢中飞出，进行扩散的现象。而婚飞则是雌雄有翅成虫为了“找对象”而群集飞舞的现象。除了白蚁，部分蚂蚁也具有分飞的现象。而婚飞的昆虫种类就非常多了，如蚊子、蜉[fú]蝣[yóu]、部分蜻蜓和蝴蝶等。

白蚁婚飞其实是一种扩张领地和繁殖后代的手段。白蚁是社会性昆虫，多数白蚁具有严格的等级划分。蚁巢内有蚁王、蚁后、工蚁、兵蚁、繁殖蚁等不同的品级，其社会分工也不同，其中只有繁殖蚁具有完整的翅膀。

每到夏天雨季前后，环境适合时，工蚁就会在巢穴附近打开一个分飞孔，繁殖蚁们在察觉附近没有危险后，就会陆续从巢穴中飞出。这时，附近的同种白蚁就像约定好了似的竞相飞出。多数白蚁喜欢在黄昏后婚飞，少数白蚁在白天雨后的中午婚飞。它们飞行能力不强，所以大多沦为了青蛙、鸟、蜘蛛、鱼等动物的口粮。

在白蚁分飞时，鸟类也忙于享受白蚁大餐。图片：wildtomorrowfund / youtube

它们飞行一段距离后便落到地上，然后开始不停地抖动翅膀，将翅膀抖落。之后，白蚁们开始找对象，互相看上对方后便会一个尾随着另一个，双双钻入地下开始建造巢穴，然后交配产卵。这对繁殖蚁就变成了“立国”的蚁王蚁后，它们会生一大堆孩子，然后建造一个庞大的白蚁帝国。

我们扎个堆儿，你尖叫什么？

在昆虫世界里，群集现象十分普遍。有些群集性昆虫只是偶然待在一起，比如许多同种的昆虫，可能会聚集在一棵它们喜欢吃的植物上。而有些群集行为则是基因控制的本能。比如社会性昆虫白蚁、蜜蜂和蚂蚁，它们终生都和其他成员呆在一起。昆虫群集的目的也不尽相同，有寻找对象、抚育后代、寻找食物、共同御敌、过冬御寒等。

成群结队的白蚁。图片：翦翳翎

一些昆虫孵化后就会分道扬镳，而另一些昆虫会待一段时间，比如一部分蝽。

一些昆虫整个幼虫期都群聚在一起，如多数蛾子、少数蝴蝶、蝽、蝇、蚊等。

群集取食的蝽若虫。图片：翦翳翎

蝽科若虫常会在孵化后短暂群集。图片：翦翳翎

群集取食的叶甲幼虫。图片：翦翳翎

而一些昆虫则会选择同一地点集中化蛹，如蝇、叶甲等。

毛萤叶甲（Pyrrhalta sp.）在榆树树干上集中化蛹。图片：翦翳翎

很多成虫会“集体****相亲”，这样更容易找到对象。一些昆虫会在特定的环境里，等候“爱人”出现，如蝴蝶、苍蝇等。另一些则会释放性信息素，吸引异性前来，如一些金龟子。


湖边正婚飞的摇蚊被灯光干扰后无法脱身而聚成一团。图片：翦翳翎

有些昆虫在成虫和幼虫期都会“扎堆儿”，可以增大生存的几率。

桑异脉木虱（Anomoneura mori）若虫和成虫群集。图片：翦翳翎

此外，有些成虫还会群集越冬。秋末，在你家暖和的屋子里，可能会遇到一大堆让你“密集恐惧症”发作的小朋友，比如萌萌哒七星瓢虫 （Coccinella septempunctata），还有俗称“臭大姐”的茶翅蝽（Halyomorpha halys）。

嘎嘣脆的油炸白蚁

那么问题来了，它们相亲就相亲，为什么要跑到咱们家里去呢？其实，它们大多是被误导的。繁殖蚁出现在家中有两种原因：

1、家中本来就有白蚁生活；2、家中开着灯，忘了紧闭门窗，繁殖蚁们成群结队进入你家，并开始相亲。

前者发生的几率不大，如果你家里有白蚁，那你打扫卫生时应该会发现，不至于到现在才突然知道它们的存在。后者则是因为灯光干扰了它们飞行的方向，它们“蒙圈”了，就会向灯飞去。

有翅的白蚁，负责繁殖。图片：翦翳翎

遇到家中爆发繁殖蚁，千万不要惊慌失措。婚飞的繁殖蚁数量很多，且头头鲜嫩多汁、软滑肥壮，它们在离巢前都把自己养得白白胖胖的，营养丰富极了。把它们抓起来扯掉翅膀洗一洗，扔到油锅里炸至金黄再撒上盐，隔壁小孩都馋哭了。它们敢在客厅里做羞羞的事情，就要让它们付出一点代价呢。

这可不是开玩笑，世界各地都有食用白蚁的记录。非洲部分地区会将白蚁制成酱，当作款待贵客的佳肴，而中国云南西双版纳人民食用繁殖蚁的历史也很悠久。如果实在下不去口，在家中发现繁殖蚁，可以紧闭门窗，然后将繁殖蚁清理出屋子。它们不会咬人，可放心用手拿捏和把玩。

莫桑比克的小朋友们在地上捡“嘎嘣脆”的繁殖白蚁。图片：TimCowley / wikipedia

白蚁的功过

每到白蚁婚飞的季节，人们就会恐慌，生怕自己辛辛苦苦付完首付的房子，隔天就被白蚁啃成筛子。各大新闻媒体也会争相报道诸如《新买房子却被“飞蚂蚁”攻占》这样耸人听闻的信息。这或多或少源于我们长期以来对白蚁的偏见。

我小时候看过的外国动画片里，白蚁瞬间就把家具、房子啃成一堆木屑，使我在很长时间内都相信，白蚁是毁灭一切木制品的大胃王。实际上，白蚁并不都是坏家伙。

白蚁吃掉了汤姆猫的椅子。图片：《猫和老鼠》，Jacob Roy / youtube

白蚁正在啃食瓜子皮，它们有时也会清理我们人类产生的垃圾。图片：翦翳翎

我们常说的白蚁不是一种动物，而是一类动物，是蜚蠊目白蚁总科昆虫的总称。白蚁的生活习性因种类而异，有些白蚁严重威胁着人类日常生活，而有些白蚁是有价值的资源昆虫。

我们熟知的白蚁的第一大危害是“食木”，会危害家具、房屋、古建筑等很多木质物品。其实，能破坏家具房屋的白蚁只占白蚁中的一小部分，它们长时间啃咬木材后会造成家具毁坏、房屋倒塌等。比如黑胸散白蚁（Reticulitermes chinensis）。这些白蚁喜欢在房屋中、古建筑内筑巢，一旦发现巢穴应当立马防治。

外出觅食的白蚁和在木头中做巢的白蚁。图片：翦翳翎

白蚁的第二大危害是掘洞。常言道“千里之堤溃于蚁穴”，这里的“蚁”指的是白蚁而非蚂蚁。它们在堤坝、河岸等处建造发达的地下巢穴，造成决堤，可能会引起淹没村庄、耕地等巨大灾害。但这类白蚁也只占白蚁中的一小部分，以土白蚁属（Odontotermes）和大白蚁属（Macroterme）的种类居多。黑翅土白蚁（O. formosanus）是危害堤坝最主要的白蚁。

有趣的是，这类白蚁并不吃木头，而是吃蘑菇的菌丝。它们从外界环境中收集枯枝败叶等植物残体，作为种植蘑菇菌丝的基质，而它们种的菌丝长大后，就会破土而出，变成蘑菇。西南地区的美食鸡㙡[zōng]，属于蚁伞属（Termitomyces spp.），就是一类白蚁经常种植的蘑菇。从这个角度看，这些白蚁又是珍贵的资源昆虫。

一些白蚁还会危害农作物和树木，它们在树内筑巢，甚至直接给树切根剥皮使其死亡，但种类不多。另外，有时白蚁还会咬坏电缆线等物品。

虽然白蚁“作恶多端”，但把所有白蚁都当成啃房子的大魔王，也是不客观的。它们作为分解者，分解森林中的垃圾，在生态系统中扮演着非常重要的角色。

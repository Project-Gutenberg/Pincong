下文是我原本放在「民族性」的答案下的。但改在這裏吧。因為就是問「邏輯」問題。任何主張立場的人都可以邏輯混亂。關鍵在「理由」。一些人的「立場」可能和你一樣，但其「理由」是極混亂的，一個主張可有不同理由，同一理由又主張不同主張。理由才是核心。主張一樣，理由不同，是分裂的種子。而邏輯混亂者，是不能溝通的，他們也不懂開解自己，和任何人溝通都是為了別人「認同」他的情緒。以下正文：

中國人最大的問題是——沒邏輯。

沒邏輯和法治一樣，是一種狀態，或外人對這種狀態的評價。
我嘗試把普遍中國人沒邏輯的原因和形式一一列出。
這些現像其實是所有人類都常見的。題主問的是「中國人」，
我就姑且稱這是中國人普遍或常見，而非獨有。

我一直都有思考人為甚麼沒邏輯，難溝通，閱讀理解能力差。其原因與智力無關，是缺乏自省和缺少一些抽象的分類方法，這就要讀哲學，特別是英美的分析哲學，模仿其方法去思考問題。要一次列出這些問題是很難的，幸好我看到了會員 snowhoof 的文章，他基本上把中國人沒邏輯的情況全犯了。我提他名字是免得他說我「暗示」。

首先，「沒邏輯」不代表智商低，因為可以是溝通能力、理解別人說話期望上有誤會。也不代表「沒邏輯」論述背後沒有「真正」的理由。一般來說，人說話背後都會有某些期望的，這些期望如果能清楚表達，外界能理解，就可以被別人認為「有條理」，有邏輯。

其二，我不是指一個人整體「沒邏輯」，而是指人會在一些事情上「沒邏輯」。也就是說，人不會在任何方面都「沒邏輯」。

其三，「自省」，是理解自己情緒流動，即時察覺、感應自己情緒、意向的流動。常見的方法是想像自己是對方會怎麼做？有甚麼感受？易地而處。

愛面子的人一般都不能察覺自己情緒流動。也很容易被操控被騙買東西。

自省能力，也與人大腦的敏感程度有關。太敏感的人很容易過於內向，容易抑鬱、沒自信。
而反社會人格一般是不會自省的，因為他們不界意別人怎看他們。這是兩個極端。
內向、敏感的人會多用別人的角度思考自己，也較有同理心。
所以，一些人的「自省」是被迫的，天生的，一些人要養成這些習慣。

對自己情緒的感應是理解萬物後，作出客觀評價的基礎技術。
可以輕易講出我討厭的東西在客觀上好的一面，不會內心隱隱作痛。
可以輕易講出我喜歡的東西的在客觀上壞的一面，也不會內心隱隱作痛。

「沒邏輯」的形式/原因：

1. 類比。

類比只是人腦確定形式、圖案的能力。
而世上沒有真正重複的事物。
分得越細，重複越少。知得越多，重複越少。

當我們對一件事了解不多時，一般會想像「這個東西」很像「那個東西」。譬如青少年看電影，會說：這套電影很像那一套電影，「我看過」，所以這電影沒甚麼了不起。他們可能把所有科幻電影都視為「一種東西」，不認為「科幻電影」之間都有差別。簡而言之，對這類的東西的欣賞能力較低。沒察覺到差異背後的意義。

欣賞能力，就是分辯事物細微差別，並理解這些差別背後意義的能力。(然而沒意義的差別，人是不能察覺到的) 。

「沒邏輯」的對話，常是其中一方察覺不到自己說的「這個東西」很像「那個東西」之間的差別，是有意義的，

人在一件事上「沒邏輯」的情況之一，是他認為「不用分得那麼細」。

知得越多，欣賞能力也越高。但這也要人主動去感應危機，帶著問題看世界，才能吸收新的差異。而且知得越多問題越多，這時，很多差別都是有意義的。

類比思維的其一害處是，把形式當做解釋。譬如問：
這課堂為甚麼亂？
答：因為這裏有五種壞學生，你看，A同學和B同學是同一種學生，C同學和D同學是同一種學生，所以，有這五種壞學生，所以課堂亂。

這就是把「亂」的形式記錄下來，作為「為甚麼亂？」的答案。

這幅畫是怎麼畫的？(成因) 
答：這幅畫上面有五個三角形，下面有三塊綠色，左邊有一點紅色，右邊有白色。這張畫是這樣畫的。
為甚麼這麼好看？(評價)
答：這幅畫上面有五個三角形，下面有三塊綠色，左邊有一點紅色，右邊有白色。所以這張畫很好看。

把這個和那個很像，那個又和這個很像，所以就會這樣的。
把形式當做解釋。
把自己看到的，種種「很像」的情況描述出來，就當作是事物的「解釋」。

類比的禍害相大家也懂，愛國宣傳常用，譬如人是國家永遠長不大的孩子。(關係的形式)

類比的問題不在說話者，因為，始終語言就是一連串的類比暗示。只是這些一連串的類比暗示(論述)是否逼近真truth和善good。(這是西方哲學吵了幾千年的東西)

類比的問題，在接收者，即我們應習慣抗拒別人的類比，這樣才能思考另一種可能性。

2. 誅心論，

誅心論來自對人的一種「本質主義」看法，而且是雙重的「本質主義」。

第一重：人都有一種「本質」，他本質好、本質壞，本質勇敢、本質虛偽。一些人本質做大事、一些人本質一事無成。這種看法是來自小孩子把洋娃娃當成朋友的能力。

人腦預測未來的方式是把任何事物都賦予一種「性格」，然後再依「性格」去預測他們。好人做好事、壞人做壞事。

收集資料、形成「性格」的判斷，再去預測其行為。人對一件事「沒邏輯」，很常是先入為主，把「性格」固定了，而固定其「性格」的是情緒、情感。所以，少自省的人，一般不察覺自己有偏見歧視。(或不認為這是問題，不用分得那麼細。)

第二重：善惡好壞對錯的評價，最終，必定都是對人性格的評價。好事，來自好人，壞事，來自壞人。

好事由壞人做出來，就不是好事。即對人的評價決定對事的評價。
也就是說，在他們腦中，人人都是虛偽的。好事，必定要來自「真正」的善心。
那甚麼是「真心」善心？由他「性格」決定。
那怎樣確定這人「性格」？我喜歡他就是勇敢，不喜歡就是魯莽。

3. 道德示範主義、家長主義、泛道德主義

a.一件事不能做，不是因為它造成實際傷害，而是因為它「示範」了不好的態度/想法，會對別人有「壞影響」、教壞人。(家長主義)
b.一件事不能做，不是因為它有害，而是因為它這樣做，他就有「壞性格」，所以不能做。(泛道德主義)

例子:
a.的例子很多，也常見。不寫了。
b. 香港的例子有：去遊行示威後，拍照放到facebook(又叫「打咔」)，是有虛榮心。有虛榮心，你就不是「真心」為民主，如果每個都有「虛榮心」，我們的理想就不能達成。»所以……你不能拍照放到facebook。

泛道德主義，要求別人說話行為要彰顯一種，和我一樣的態度(堅毅、勇敢、敵視等)，否則，就「顯示」你的「本質」不好，或者會慢慢變壞，會壞了我的大事。(文革常見)

道德示範主義、家長主義、泛道德主義在文革很常見。來自儒家社會對「品格」的滑坡推理。中國人一罵人就是用這一招。

4. 偽善論。 有很多種變體，總之任何人任何動作都是偽善的，這又回到第一點：人的本質主義。
詳情後補。

5. 資格論。
「你的回答可以在國內考很高分，你沒資格反共。」snowhoof 的資格論、泛道德主義。
資格論是貢獻論，貢獻大的人，應該做大佬。

這種貢獻是甚麼？是付出。
我付出很多，你沒資格評論。
說話者意見的「份量」與其付出成正比，而不是他意見的理據。

「付出」多少，在不分離情緒的情況下，又變成主觀的付出。
你覺得自己付出多少就是多少。

資格論 100%來自情緒。沒甚麼道理可言。

資格論第二種是傳統的論資排輩。不尊敬前輩，「顯示」你沒有好的「性格」，所以你會「學壞」。又是回到人格的判斷上。總之中國人腦中就是「人、人、人、人」。

6. 情緒和理由混合。

這是中國人沒邏輯之母。

如前面說，一個主張可有不同理由，同一理由又主張不同主張。理由才是核心。主張一樣，理由不同，是分裂的種子。而邏輯混亂者，是不能溝通的，他們也不懂開解自己，和任何人溝通都是為了別人「認同」他的情緒。

中國人由於不能，不懂、不習慣表達情緒，所以和你討論問題不是「我的看法有以下理據：A, B, C, D」，
而是「我的討厭、厭惡、憎恨之情，是合理的，你應該肯定我的這種情緒，因為A, B, C, D」

「我討厭的東西，你也應該討厭，因為A, B, C, D。
你做不到？再看一次，理由是「A, B, C, D，
我再重複一次，理由是「A, B, C, D」，看，你應該討厭它吧。」
這不是討論，是強迫推銷，人家不買帳就開始「這個人沒資格如何如何」，人格問題就來了。

這也是人閱讀理解能力低下常見的原因。
表面是在「說服」別人，但一直就是為自己情緒辯護，寸步不讓。

也就是說，中國人表面在「討論」，實際是在迫使對方「認同」自己的情緒是「合理」的。
這是因為，
中國人必須對 應該愛的事物產生「愛」的情緒，
中國人必須對 應該恨的事物產生「恨」的情緒，
自小沒有愛的自由，沒有恨的自由。這也是因為中國人獨特「責任」觀。中國人不輕易背負任何責任，責任都是自小被強迫背上的，情緒上認可，就等於要背上責任了。

有人说：“有国才有家”就跟“有墙才有砖”一样——墙是砖砌的，明明是先有砖，后有墙，怎么能说有墙才有砖呢？

其实这个问题没那么简单。如果我问你：世界上为什么会有砖呢？你可能会回答说：砖厂生产的啊！但是砖厂为什么要生产砖呢？为了砌墙或者盖房子……因此，说“墙”是“砖”存在的原因，也是能够成立的。

我们通常认为“原因”总是发生在“结果”之前，为什么在“砖”和“墙”的案例中，原因却出现在结果之后呢？

这是因为我们平时所说的“原因”和“结果”，其实有几种不同的含义。

例如，我们问一个人：小明为什么死了？答：因为他从楼上跳下去了。这里说的是时间顺序意义上的因果关系。

我们再问第二个人：小明为什么死了？答：因为万有引力。这里说的是还原论意义上的因果关系。

我们再问第三个人：小明为什么死了？答：因为他要自杀。这里说的是目的论意义上的因果关系。

这几种回答都能成为对“小明为什么死了”的解释。其中，目的论意义上的因果关系就是一种逆行因果关系，在这种因果关系中，原因出现在结果之后。

人类制造各种东西都是为了某种目的，解释某种人工制品的存在，显然离不开人们制造它的目的，而这个目的显然只有在产品生产出来之后才能实现。因此，我们在讨论“砖”这种人工制品的时候，逆行因果关系是可以成立的。
那么，“有国才有家”这句话错在哪儿呢？

无论是个人还是家庭，都不是为了国家而存在的。康德告诉我们：把人当成目的，不能把人当成手段。人和砖头一类的人工制品的不同之处在于，人存在是为了自己，而不是为了任何其他的目的。把个人（或家庭）和国家的关系比作“砖”和“墙”的关系，是一种不当类比。

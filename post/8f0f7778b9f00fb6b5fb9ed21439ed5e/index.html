# 此圖非新聞，僅為他人的二次創作 #

圖片是一個關係不錯的朋友傳給我的。
不是分享新聞，是他拿來諷刺我對香港抗爭的支持。(補充一下，避免有人誤會成宣傳假新聞)

他本來就非常反感香港的抗爭，我花了滿長時間才說服他，勇武派是民意被漠視的結果，說香港抗爭者“暴徒”是倒果為因。
然後他今天傳來了這張圖。
當下真的心情複雜，失望、難過……還有許多的一言難盡。感覺即使大家努力了很久，聲量還是如此微弱，有些事情還是不被人們理解，用偏見扭曲一切善意。

香港跟美國完全是兩個不同社會！
「公民擁槍權」對美國警察文化影響非常明顯，他們行動是在假設「所有人都可能下一秒掏槍」的前提下決定的。我自己不怎麼愉快的體驗過一次這種文化差異，印象深刻……至今不能忘。

    ＃在美國，公路上開車被警察攔下，駕駛不能「主動下車」！ 
那次真的沒有多想，認為自己被攔的莫名其妙，有點情緒，就直接下車準備問人，喜提了一份 經 典 美 式 壓 車 制 服 ，沒有很重手，但足以嚇爆我一個從沒惹過事的小慫蛋。QAQ
差點嚇死！這輩子都沒想過！雖然知道不能有刻板思維，但作為女性、又是亞洲面孔，已經習慣了自己是不會碰上暴力待遇的群體，突然被公權力執行者壓制對待真的整個人嚇飛了。警察行動是完全以你下一秒要開槍的假設來進行的，那種狠辣，是我們平時根本體驗不到的人性陰暗面。
(完全可以理解壓頸案受害人被嚇出心臟病的狀態  (小聲

我是支持抗議警察暴力的，受害人死前呼喊了十五次的無法呼吸，全場所有警察沒有動作，發現他沒脈搏了還持續壓上了數分鐘，行為徹底漠視了一個人的基本生存權。
雖然偏激的暴力警察文化是全民持槍的結果，但那種狠戾文化已經威脅到了每個人。
公民武力持有讓公權力執行者反而視公民為敵人，諷刺又可怕。

心情好複雜，也不知道自己想總結什麼，就當我隨便亂寫吧。
今年真是一場大型的黑色荒誕劇本，但願是個喜劇吧……

是，準確來說我朋友算我半個前老闆。
老一輩臺灣人對香港有種隱形厭惡(?
他的年輕時代正好是香港繁茂，臺灣還在做代工廠掙扎賺錢的時間。當時香港公司老闆聽到「臺灣商人」，第一反應會是搞山寨假貨、沒有商業信譽(不過那時台灣真的專門做山貨)，跟他們打交道多半要受點質疑、受點氣，放到今天就變成了謎之怨念。

再來是他超反感學生抗爭。
之前被直接抹除的兩岸服貿條例，影響到很多生意人的規劃，其實像影視娛樂這些，大家網路早就互通了，開放雙方貿易完全是沒什麼衝擊的。(你信臺灣現在一年只能"合法進口"十部中國電影嗎？還要用抽籤這種鳥方式……)
但2014一場太陽花學運，在學生主張拒絕開放的潮流下，兩岸剛有起色的貿易環境改善被一刀切，什麼都不剩。
做老闆的只能認賠，收掉了不少本來有興趣發展的領域。從此他老人家就恨透了學生運動，認為學生意見團體既幼稚又理想化。
連帶著，在香港抗爭漸起後，以學生為主力軍的雨傘革命他也看不順眼，實習生在辦公室座位放香港文宣小貼紙，直接整組團隊被檢討。(噢 我就是放小貼紙的白目實習生之一 (小聲

可能聽起來很鄉愿，但跟老闆深談後，我是能理解他對香港抗爭的不喜，也尊重他。
雖然跟大佬意見相左，但整個實習工作經驗仍然收穫很多，老闆他在個人立場之外待人平等，不認同我們的想法，但願意去以朋友相稱，理性對話。

我們這些放文宣的小刺頭後來都很喜歡他XD
當時的實習生群組也就保留到了現在，幾個活躍者時不時分享一些訊息，也很有趣。
然後……就是老大他今天早上貼了這麼一張諷刺圖，群組沉默到下午才有人回了一句 “美國社會不需要五一” 。
很難受阿，當朋友一樣的長輩，也不是個壞人，偏偏在這事上拿個人偏見來看待香港。

从心如我。
跑來品蔥寫了一堆。
還是沒敢去當面懟大佬。(望天

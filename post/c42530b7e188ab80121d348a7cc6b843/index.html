#唐山 打人# 从法律层面说一下这件事。

一：那个绿衣男过来搭讪并拿咸猪手摸白衣女孩，女生推开他的手，并说：干啥呀？有病呀你？

这个时候如果事态没有继续恶化（绿衣男良心发现没有进一步实施侵害行为），此时女生如果报 j，警 c 来了大概率是和稀泥的。 特别是男性警 c。男 · 警 c 会说：“就摸了两下，人家就是想认识你，别大惊小怪”、“没有摸特殊敏感部位，只摸了你后背两下、不算猥亵”、 “别浪费我们警力”。

结局百分之九十九的可能是，口头调解一下，让双方在出警记录上签字就完了。

如果女孩特别较真，有时间和警 c 缠斗，拿着视频证据，天天去派出所找警 c 给个结果。
警 c 大概率连 “不予立案通知书” 都懒得开。就算立案了到最后也是 “情节轻微” 不起诉。

【也就是说，那个女孩如果被摸了后背或者脸，大概率只能自己忍受这恶心，法律层面咸猪男得不到任何惩罚。】

二：可是事情恶化了，女孩儿推开了咸猪手，男的不仅没有走开，还变本加厉打了女生一巴掌。 如果此时，女孩不想让事情恶化，那女孩就得忍着，默默走开，报警。

【这个时候如果女孩报警了，男的可能会是 “寻衅滋事罪” 或者“故意伤害罪”。看那一巴掌有没有把女孩打成轻伤以上的程度吧，一般很难构成“故意伤害罪”，大概率是“寻衅滋事罪”。 又或者构不成刑事犯罪，也就定性为违反了《治安管理处罚法》，男的被行政拘留几天就得了。】

可女孩也是人，是一个正常人。一个正常人，我们无法期待她在被性骚扰后又被打，还能情绪稳定地起身离开走到一个安全安静的地方报警。——即使她做到了，我们还得祈祷咸猪男不会离开犯罪现场。

三：女孩在被摸又被打后，反抗了。她拿酒瓶子反击了。 后续女方三人与一群咸猪男产生了冲突。

任谁看了视频都会认为，女方是正义的一方。但在法律上不是。

这时候如果警察来了，大概率定性为【互殴】。

互殴怎么担责？双方都得担责。
男的把女的打成重伤，那男的就承担【故意伤害——重伤】的责任。
女的把男的打成轻微伤，那女的就承担【故意伤害——轻微伤】的责任。

所以这件事，不管发生到哪一步，女生在法律上是很难 “伸张正义” 的。

上文，我很多处都用了 “大概率”，因为这事现在上热搜了，所以有可能会出现“小概率” 的通报结果。 但，现实生活中，不是每个被性骚扰的女生都能上热搜。

那些男的看起来就是烂人，即使被判了刑，我猜测，他们也会摆烂，在民事赔偿上不一定到位。 女生被打的满身是血，住院治疗，大概率还得自己掏钱。

法律从来不保护女性。 这就是现实。

写给百度网盘刚需者从零开始的使用方案：将百度网盘客户端安装在 Sandboxie（沙盘）内，运行时与系统相隔离，同时管控后台运行，防止偷跑流量。此方法同样适用于微信多开、调教腾讯课堂与 YY 等国产软件。

3. 下载百度网盘安装包，右键单击选择 “在沙盘中运行”，选择刚才创建的沙盘并确定运行。随后的安装过程与平常完全相同，但光标移到程序窗口顶部会看到一圈黄色边框，这就是程序运行在沙盘中的标志。

4. 在 Sandboxie Control 或者托盘图标菜单中找到刚才创建的沙盘，选择 “浏览保存内容”，即可找到程序的真实安装位置，位于沙盘的隔离区域，与你安装时选择的位置相对应。

位于沙盘的隔离区域

在同一个菜单选择 “快速恢复”，选择将这个快捷方式恢复到相同文件夹，即可在本机桌面快速创建沙盘内百度网盘的快捷方式。也可以从上述安装位置选择主程序 “BaiduNetdisk.exe”，发送快捷方式到桌面。完成其他设置后，只需要从这个快捷方式启动。

6.（可选）由沙盘的运行机制可以知道，事实上沙盘内的百度网盘仍然可以读取整个硬盘上的文件（只是写入被沙盘限制），对此可以选择禁止沙盘内的程序读取敏感位置，但这样会影响从被 ban 路径上传下载文件的功能。另外，注意阻止访问设置的优先级高于其他所有文件访问设置，不要限制百度网盘安装位置、文件下载位置、必要的系统文件路径以及它们的父目录，防止影响百度网盘的运行。如果第 3 步安装百度网盘时选择了没有敏感文件的分区，这里会方便许多。

具体设置请自行斟酌。仅作为参考：我 ban 掉了图片和文档文件夹、OneDrive 同步文件夹、Aria2 下载文件夹、一个敏感分区和其他存有敏感文件的路径。以下有两种方法：

(1) 在沙盘设置 → 资源访问 → 文件访问 → 阻止访问，点击 “添加”，选择你想要禁止读取的路径并确定。每次只能添加一个路径。

(2) 在 Sandboxie Control 窗口 → 配置 → 编辑配置文件，可以手动编辑配置文件。找到之前设置的沙盘名称（通常在文档最后），以 “ClosedFilePath = 路径” 的格式添加到配置文件最后，每行一条，编辑完成后保存关闭。

下面举几个例子，例如要禁止读取整个 E 盘，像这样写：

ClosedFilePath=E:\\

例如要禁止读取 D 盘下的 Downloads 文件夹，像这样写：

ClosedFilePath=D:\\Downloads\\

例如要禁止读取 Windows 图片文件夹和文档文件夹，像这样写：

ClosedFilePath=%My Pictures%\\
ClosedFilePath=%Personal%\\

指这两个文件夹

7. 百度网盘虽然可以选择不加入 “用户激励计划”，但没人保证取消勾选后它不会偷跑。在沙盘设置 → 程序停止 → 先导程序，点击“添加程序”，选择“baidunetdisk.exe” 并确定，即可在退出客户端后自动杀掉此沙盘内的所有进程，以绝后患。

如影响运行请取消此选项

如果百度网盘的不能正常运行和升级**，请取消** “降低管理员和超级用户组的权限” 选项。

为何网盘成本高？用户不能承受之重

百度网盘 “用户激励计划” 引起的热议和反弹，也有不少是关于背后原因的反思。一个主流的观点是网盘运营太难。

据了解，个人网盘确实是运营成本高昂的门槛的互联网产品，主要包含硬件、电费和带宽，平均下来，0.15 元 / Gb 每月，也就是说 1T 空间一年的成本是 1800 元。

个人网盘的云存储成本

从国内网盘发展看，确实存在网盘厂商的短视。2012 年左右个人云存储兴起，大大小小的网盘跑马圈地，不计成本的提供超大空间甚至不限空间，巨量用户涌进来，运营成本也直线上升。厂商一厢情愿的变现不理想，无论是广告模式还是会员收费，都是入不敷出难以为继，这也导致了在 2016 年开始的网盘关停潮。

时不时针对免费用户的调（缩）整（减）空间、限制传输速度，包括现在百度网盘用户奖励计划占免费用户的 “好处”，背后都是这个原因导致的。

反观国外的网盘市场，以 Dropbox、Google Drive、OneDrive 为代表的，市场比较理性，为用户提供的免费空间很有（吝）限（啬），但不会被限速，更多的空间需要付费。厂商能够比较好的控制运营成本，重点发展付费用户扩大收入，走上一条良性、可持久的 “高质量” 发展之路。

但实际情况是：每月 1000 积分的重要条件之一是每日持续贡献 1MB/s 带宽，关键词在于 “持续”，但根据极客君的实测，在绝大多数的情况下，1MB/s 的上传速度都是无法达到的，这一速度并不稳定，时而为零，时而上百 KB/s。而能拿到的积分也是和上传速度息息相关的，如果不能稳定 1MB/s 的上传速度，每月 1000 积分自然无法保证。除了持续 1MB/s 的上传速度之外，关键条件还有每日 7 小时在线时间，这样最终才能获得每月的 1000 积分。

国内网络运营商提供的绝大多数宽带都是上下行不对等的模式，比如标定 100Mbps 的宽带，100Mbps 是下行速度，也就是 12.5MB/s 的下载速度，而上传速度一般在 20Mbps，也就是 2.5MB/s，百度网盘最高 1MB/s 的上传速度就占到这个速度的近一半。

如果是 50Mbps 的宽带，上传速度一般在 1.25MB/s，百度网盘的后台占用上传带宽会明显影响到上网体验。

NO.1454- 舶来蔬菜的命名

作者：鸡腿周

 制图：孙绿 / 校稿：猫斯图 / 编辑：养乐多

中国讲究民以食为天，重视瓜果蔬菜五谷杂粮均衡膳食，用不同的原料调出丰富的口感和口味，满足不同人的口腹之欲。

东北名菜——五谷大丰收

（图片：图瑞 / 图虫创意）▼

但今天我们能在餐桌上看到的食物，并不是被某个神从创世之初就播撒在东亚大地上的，而是在千百年的文明历程中，经由中国商人不断引进，并在本土农民的驯化改造中，逐渐适应了中国风土，成为桌上佳肴的。

中国地域辽阔，地形和气候类型都很多样化

很多外来物种都需要进行本土化改造

（四川省-广安市-岳池县-伏龙镇梯田）

（图片：何永平野马 / 图虫创意）▼

这些食物名字中的“胡”、“番”、“洋”字，就已经暗示了这段历史的存在。

欧亚大陆物种大转移

中国从域外引进外来物种的历史，其实从先秦时代就已经开始了。但是由于那一时期无史可考，且以当时的交通运输和农业科学条件来看，引进物种虽好，却很难在真正意义上得到普及。这一时期的物种交换也主要以考古证据的形式留存于世，很少有时间确切的文献记载。

古代世界，对植物和动物的驯化是最基础的科技

世界上早期文明往往都有自己单独驯化的农作物

而在交流过程中，同纬度的动植物资源最容易移植▼

但随着东亚大陆在秦汉完成统一，并依依西望，向西域投去了好奇的目光。尤其是张骞通西域之后，西域诸国与中央王朝的商路被打通，大量后来被中国人习以为常的食物，就是这个时候被从欧亚大陆深处带到东亚的。其中既有始见于中东的核桃，也有最早见于欧洲的大蒜，甚至还有原产于非洲的芝麻。

这些现在常见调味料

很多都是西域和南亚、东南亚的舶来品

（图片来自：图虫创意）▼

对于中央王朝来说，这些物种皆是自西向东入华，故而大多带一个“胡”字，正如《翻译名义集》所说：“自汉至隋皆指西域以为胡国。”比如核桃到现在还被称为“胡桃”，大蒜芝麻也在那时被称为“胡蒜”、“胡麻”，黄瓜豌豆则是“胡瓜”、“胡豆”。

（北魏-丝绸之路-胡商俑）

（图片：Wikipedia Commons）▼

而打通了西域的张骞，就被人们想当然地认为是这些外来物种引进之父，《本草纲目》上就很多见“张骞使西域始得种”的记载。但李时珍是明朝人士，亦非历史大家，对汉史的造诣自不如其药学造诣，这些考证在汉朝正史上也不见记载，很可能只是医者之间的讹传。唯一能确定的是，这些物种确实已经在汉代出现了。

比如很常见的黄瓜，曾经就被成为胡瓜

其实也是从西域传入（可能西汉时期）

（图片来自：Alex /  Adobe Stock / 图虫创意）▼

所谓一方水土养一方人，也同样养一方物种。欧亚大陆能在公元世纪完成数量如此庞大的物种交换，主要还是得益于它横向的版图形状，使得物种适应新家更为迅速。但要让远道而来的生物扎根，人的努力也是很重要的，农民需要流十分的汗水，还要得到科技加持，才能成功。

而且不同农作物之间移植的难度也差别很大

比如苹果就比较难

（果树需要更长时间的培养才能结果

且驯化果树本身也是最晚）

（图片：deposit / 图虫创意）▼

就比如现代北方人的主食小麦，其实在先秦时就已经从两河流域逐步传入了中土。可它始终不是主流作物，在主粮市场上无法与当时北方的主食粟（小米）竞争，即使西汉董仲舒力推这种“嘉禾”也不见效。但在高质量的铁农具在魏晋时期普及之后，北方农民找到了种植冬小麦的方法，这种谷物的年产量开始超过小米，它终于适应了中国的主流需求。

1967年-北京郊区小麦丰收

（图片：《人民画报》/ Wikipedia Commons）▼

到了唐朝，小麦已经逐渐取代小米成为了北方人的主食。不过蒸煮麦粒实在不好吃，于是聪明的中国人又用处理大豆的方法对它进行磨粉再蒸煮，终于诞生了花样繁多的中式面点。小麦作为引进物种高效利用的典范，从此改变了中国人的餐桌。

从麦粒到面粉，这种基础性的技术革新

在那个依靠人力畜力的年代，也算是巨大进步了

（图片：dashiyiyuan_vip / 图虫创意）▼

新大陆来的新客人

从秦汉开始的欧亚大陆物种大交换，其实陆陆续续进行了1000年，直到宋代还有一些补充性的物种在向东方传入。但时过境迁，人类对地球的认识范围正在逐渐扩大，欧亚大陆，乃至非洲大陆上的物种传播已经接近尾声，新大陆上的物种正在等待着人类去揭开它们神秘的面纱。

比如玉米和土豆这两种关键高产主食都来自新大陆

作为原产地，当地的玉米也是品种繁多

（图片：shutterstock@Jose de Jesus Churion Del）▼

与中国的元明代同期，环大西洋航线已经成为了西方海洋国家的囊中之物，新旧大陆之间的物种交换，也在这个新的海洋时代逐步实现。随之，以西方强国在东南亚的殖民地为跳板，这些新物种又逐步传入了中国。

我们日常见到最多的则是黄色这种

（图片：在路上212 / 图虫创意）▼

这一时期进入中国的物种，大多走海路从东南方向进入，因此名字中大多带一个“番”字或“洋”字，在一些内陆地区也可能带上“海”字。

比如西南人最爱的辣椒，在本地方言里就是海椒，而当时其他地区的人们管它叫番椒，说出了这种食物从外国番邦沿海路传入的真相。类似的食物名称遗痕还有很多，比如土豆这种美洲土著食物在西南叫洋芋，在江南叫洋山芋，都是用个“洋”字套已经认识的物种。而有意思的是，吴语区的山芋也是外来物种，学名番薯，同样是中美洲物种。

红海椒，绿海椒，白海椒

辣椒一开始是作为观赏作物传入中国的…

（图片来自@MICHEL / 图虫创意）▼

这些新来的物种，现在的我们也都已经很熟悉了，有可作为主食的土豆、玉米，也有作为菜肴的西红柿、西葫芦（“西”字也是个暗示），还有大量“番”子头的热带水果，比如番石榴、番荔枝、番木瓜等等。

感谢新来物种，炕洋芋实在是太好吃了

（图片：图虫创意）▼

新大陆来的新客人，实实在在地解决了旧大陆主人的温饱问题。

由于以上这些美洲食物中有不少源自中美洲的高原农业区，因此天然适应山地环境，当时被欧洲人引进就是为了提高土地开发效率，在传入东亚之后也承担了山地开发的重任。明末清初，中国地方大量开荒，农民一波又一波从平地向山地冲锋，东亚人口也随之再次膨胀，中国餐桌的花色也更多了。

毕竟，中国有太多的丘陵地带可以开发

但新作物的强大适应性也使这种开发强度越来越大

接着就是更多的人口和更多的开发，形成一个循环

（图片：shutterstock@Anton Balazh）▼

吃货的力量是无穷的，随着各种吃法被开发出来，农民在供给端也努力地开发出了一些适应中国口味的亚种，比如用来炒菜的硬质土豆、可以生吃的甜口软糯的番茄、调味更加丰富多元的辣椒等等。

众所周知，中华民族在农业问题上自带天赋

（山东-寿光市-蔬菜科技博览园-小西红柿）

（图片：秋影随风 / 图虫创意）▼

这是一项漫长的系统工程，直到今天都依然是中国农民和农科工作者在投身的工作。从育种、驯化、本土适应，再到烹饪与利用，中国人从未忘记更好地利用舶来物种为自己服务。

来点现代口味

对于中国来说，到了清代中期以后，物种交换其实已经比较完整了。来自新旧大陆上的不同物种——无论是适合温带还是热带生长，都已经被幅员辽阔的中国集齐。

但新农作物种的引进，却还在继续。只不过最后一批逐渐进入的物种主要是补充性的瓜果蔬菜。这些瓜果蔬菜除了被端上餐桌，还有一些经过人工本地化变成了农副产业原料，比如山东烟台的特色酿酒葡萄蛇龙珠。

烟台和葡萄的不解之缘，还要说回1871年雅加达法国领馆的一次酒会。与胡雪岩齐名的南洋巨商张弼士前去赴宴，结识了一位法国外交官，得知对方早年到过山东烟台，看见那里漫山遍野生长着野生的葡萄。用便携机器榨汁酿酒以后，发现味道还真不错。

说者无心，听者有意，张弼士多年后亲身赴烟台考察葡萄种植和水土情况，发现烟台确实适合种植酿酒葡萄。

烟台葡萄产区位于北纬37.5°左右，与法国波尔多产区相平。临近渤海的地理位置，让烟台全境处于暖温带湿润季风气候区当中，相比于同纬度的内陆地区，气候更加温和，降水充足光照好，适合优质葡萄生长。

烟台气候温和，自然条件优异，可谓风水宝地

相传八仙就是在这里过海的

（图片：健忘的行摄世界 / 图虫创意）▼

此外，烟台的地形多为丘陵和山地，土质较粗，这也就意味着良好的透气性和丰富的营养。植物根系在这里不需要耗费太多能量，就能找到充足的有机质和矿物质营养，并得以茁壮成长。

因此，在1892年，张弼士创办烟台张裕酿酒公司，开启了中国工业化酿造葡萄酒的先河。当年张裕公司为酿出好酒，特意从葡萄酒旧世界（法国、意大利）引进了124个优良葡萄品种，共25万株苗木，最终在华夏土地存活并有稳定表现的仅有25种。其中从中择优育苗，培育出了一种与旧世界葡萄种性状不同，但品质同样稳定杰出的酿酒葡萄——蛇龙珠。

张裕酿酒公司原址▼

这是一种用于酿造中高档干红葡萄酒的葡萄品种，江南大学生物工程学院与张裕技术中心合作进行的研究表明，蛇龙珠中检出构成葡萄酒品种香气重要成分的萜烯类化合物多达26种，而在赤霞珠（Cabernet Sauvignon）、品丽珠（Cabernet Franc）、梅洛（Merlot）中分别检出的此类成份分别为17种、15种、16种，蛇龙珠的香气成分远远高于后三者。

更重要的是，和历史上被引入中国的舶来物种一样，已经在百余年育种下高度本土化的蛇龙珠，在香气复杂之余高度匹配中国人的口味。以它酿造或者混酿的葡萄酒，突出的是果香，口感柔润，单宁的涩味被很好地调和，不会散发出中国人不习惯的涩嘴味。

1931年，张裕酿酒师以蛇龙珠为主要原料，酿出了中国第一瓶干红葡萄酒——张裕解百纳。国际葡萄与葡萄酒组织(OIV)前总干事让•马里•奥兰德(Jean-Marie Aurand) 在品尝了张裕解百纳干红后就称赞道：“蛇龙珠是一个堪称完美的葡萄品种，也是有中国特色的葡萄品种。”

点击观看蛇龙珠国漫视频

《一颗葡萄的奇幻漂流》▼

可以说，烟台的蛇龙珠，续写了延续数千年的中国物种引进史的近代部分，将葡萄酒旧世界的经典物种成功本土化了。

作为中国风土表现最好的葡萄品种，蛇龙珠让中国葡萄酒更自信。

在世界葡萄酒舞台上，张裕用蛇龙珠酿造的各款葡萄酒接连斩获16项国际大奖。去年《Drink Business》举办的“全球畅销葡萄酒盲品赛”中，在来自五大洲7个国家并覆盖了“旧世界产区”和“新世界产区”的多款全球畅销葡萄酒当中，张裕特选级解百纳干红斩获全球TOP5的佳绩，据赛后公布的成绩汇总表显示，其中有3位评委给张裕解百纳打出的分数超过90分。

当年引进并成功驯化了蛇龙珠的张裕公司也一直在倾心扶持这种中国特色的酿酒葡萄。2016年5月25日，张裕公司倡议设立”世界蛇龙珠日”，旨在向全球爱酒人士普及蛇龙珠的东方魅力，推动中国葡萄酒走向世界。不少国际葡萄酒大咖都指出，蛇龙珠已然成为中国葡萄酒的一张名片。第四版《牛津葡萄酒词典》也已经把蛇龙珠作为中国的特色葡萄品种单列为一个词条。

中国的物种引进和本土化历史经历了数千年传承，历史悠久，其中的明星品种值得这样的肯定。在未来，随着中国葡萄酒的崛起，蛇龙珠必然像金芬黛（Zinfandel）之于加州，马贝克（Malbec）之于阿根廷和佳美娜（Carménère）之于智利一样，成为东方葡萄酒风味扛旗者。

现在，让我们从蛇龙珠开始，爱上中国葡萄酒！并一起为这个极具中国特色的葡萄品种点个“在看”吧！

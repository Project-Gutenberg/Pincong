美国没有「屏蔽」的概念，中国人发现在美国无法访问的中国网站，那基本上都是网站自己拒绝美国 IP 访问的。

有些是版权原因，这很正常，因为同一份内容在不同国家和地区有不同的版权人，一部电影在中国的版权人可能是一家中国公司，这家公司可以允许其中国用户播放，但外国用户就不好意思了。外国用户能否看，中国版权人可无法授权，必须是所在国家的版权人授权才行。这是全世界通行的法则，Netflix 上很多电影一些国家能看另一些国家不能看就是因为这个问题。Netflix 为此出资拍片，自己是全球版权人就可以授权全球用户看。

有些是因为内容管制。中国用户内容管制相对成熟，知道你手机号、微信号、IP 地址，不知道你是谁也能用这些信息把你供出去，你也不敢乱发什么。国外 IP 进来乱发内容，人是抓不着的，出了问题只能由网站承担责任，有些网站不想惹麻烦就直接拒绝国外 IP 访问。

反向的例子也有，很多年前百度日本上线时就拒绝所有中国 IP 访问。这完全不是被墙，只是开放访问的话中国 IP 就会去试一下百度日本能搜索什么百度中国搜索不到的信息，最终一样被墙。反正要被阉割，不如自己动手自宫。就算我们作为百度中国员工，也不是想看百度日本就直接能看的。

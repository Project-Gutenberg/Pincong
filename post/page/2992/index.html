干净的运营商。确保不被信任的【中华人民共和国】运营商不与美国电信网络互联。这些公司对美国国家安全构成威胁，不应提供往返美国的国际电信服务。 Exchange，降低中美之间的网络速度，而且提升互联互通成本，对使用美国服务器翻墙的各位也是个打击，，，（话说，你们知道我在说啥么，看看中国电信AS4134，中国联通AS4837，中国移动AS58453的peer列表，看看把美国的都去了会有什么后果） 如果是我，我会要求美国运营商严查来自特定ASN的BGP通告，而且尽快启用BGP上的安全特性，每条BGP通告都要求数字签名认证，防止BGP劫持，，，（一般【大运营商】都不喜欢部署BGP安全特性，因为实在太麻烦了，不过在国家安全面前，防止网络流量流向不该去的地方，不部署也得部署） 另外，不是把天朝等敌对国家一刀切就完事了，必须加强运营商内部人员的安全培训，防止社工，，， 2. 干净的应用商店。从美国移动应用商店中删除不受信任的应用程序。中国的应用程序威胁着我们的隐私，扩散病毒，传播宣传内容和虚假信息。美国人最敏感的个人和商业信息必须在他们的手机上得到保护，以免被中共利用和窃取。 这条问题不大，把天朝应用从商店下架，或者至少明确标注。据我观察，无论是苹果App Store还是谷歌Play Store都有大量看起来是中国人拼音的个人开发者，真要好好审查一下了，，，（不能一杆子把天朝人全部打死，但是天朝不少人开发应用的时候使用被篡改的SDK，要不然苹果XCodeGhost是怎么闹出来的） 3. 干净的应用。防止不受信任的中国智能手机制造商在其应用商店预装或以其他方式提供下载受信任的应用。华为作为中国监控国家的一个分支，正在利用美国和外国领先公司的创新和声誉进行交易。这些公司应该从华为的应用商店中删除他们的应用程序，以确保他们没有与人权侵犯者合作。 分两部分：一是防止华为之类的天（军）朝（方）公司预装谷歌之类的应用，二是要求美国公司应用不得在华为之类的应用商店上架，我币圈人支持一个，，， 4. 干净的云服务。防止美国公民最敏感的个人信息和我们企业最宝贵的知识产权，包括COVID-19疫苗研究，在云端系统上的存储和处理，被我们的外国对手通过阿里巴巴、百度和腾讯等公司访问。 这条我大力支持，就算是美国的云服务公司，也不知道与天朝在做什么PY交易。通过审查天朝公司的投资，确保内部不会泄露数据。通过强制性安全审查，确保不会被外部黑客攻陷，，， 5.

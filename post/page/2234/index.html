國小到國中不用考試升學的階段，學校內會有各種不同的特殊班級。特教班是專為身心障礙者開設的班級，班級內只有身心障礙者，但其他班級都是正常孩子，不會失去互動的機會。資源班則是為症狀更輕微，在一般班級和普通孩子一起上課的身心障礙孩子設立的，每天都會抽一到二堂去上課。 除非障礙程度太嚴重，學校無法負荷才會送到特殊教育學校，過著幾乎無法與一般孩子互動的學校生活。 困在国内这么久，主要是疫情走不掉，现在终于再次肉翻出来了。谈谈这一年在墙内亲身经历的感受吧。想到哪儿写到哪儿 1.经济下滑，但还没到伤筋动骨的地步 店铺大量倒闭是肯定的，尤其是餐饮业，ktv，健身房，旅馆酒店，旅行社，机场免税店等 外企撤离导致不少人失业，还没撤离的一些行业遭遇大规模降薪，比如汽车业，像通用这种公司的员工从疫情开始降薪到60%，至今还没涨回来。很多人都想转去特斯拉，因为只有特斯拉正在往中国进。 大量公务员事业单位也被减薪，有些单位今年初和去年底时一度被欠薪。 但是据我观察，国内目前大多数人正常维持生活还是没问题，只能说不耽误他们吃喝养家糊口。也可能是某些媒体揣测的，土共在这一波疫情成了得利者。 总之目前的经济形势，从民生上还看不到土共要垮的迹象，社会秩序还能运转。也是因为没影响到生活，脑残的粉红粉蛆，爱党P民还是占大多数。即使有些人对体制不满，他们还是觉得党总是好的，国还是要爱的，只是下边有些贪官该换换 2. 没感受到粮食危机 youtube各路自媒体一直说中共粮荒快要来了。我一直没少买屯粮，但这一年来据我观察粮价波动不大，猪肉确实还相比去年非洲猪瘟的时候有所降价。只不过有些小城市的工资就3，4000千，吃肉的机会少一些而已。其他米粮主食什么的几乎没啥波动。 这个问题我也很奇怪，秋粮大幅减产是肯定的，土共外汇缩减买不到粮，其他国家因为疫情粮食生产也受影响因而不愿意出口。基于这些原因我也觉得粮食会出问题，但是目前来看还没有这个迹象 3. 外汇管制更加严厉 有几个大银行，建行农行什么的，自动提款机提款结束后，经常被问如果您有境外转账的业务，是否要取消。取钱后要结束了如果点的快，很可能就点上了。 每人每年只能换5W美元等值的外汇，每周如果去银行取外钞，最多不能超过7000美元，外汇转账每年不超过5W美元 数字货币全面实施后，据说数字货币无法兑换外汇 4. 因为举报检举频繁，事业单位和高校风声鹤唳 大学里面频繁有书记和教授落马。老师上课时有监控，经常会有退休的老师担任督导，定期查课。上课时只要讲到和课程无关却与政治有关的，被学生举报，轻则警告重则停职。另外很多时候由于内斗，举报也是让下面的人上位的手段，今年已经有多个名校的党委书记落马了。其他院长一级的更是不计其数 事业单位也不能幸免。以前都知道没人吃死工资，利用手上的权利用项目，招标什么的去收礼收红包，或者公款私用。现在因为举报成风，说真的，这方面还真的收敛了，挪用公款或者收礼的事儿确实变少了，这个可能大家不太相信。不过并没有给老百姓带来啥福利。这样的结果就是给公务员一种感觉，我干多干少就这么点儿钱，那我还不如不干。所以尸位素餐的极其多，行政效率依然缓慢。去政府部门办事还个个跟大爷一样 5.清醒的人依然占极少数 我本来以为这次疫情下各行各业几乎都受影响，很多人会反思。没想到，大部分人还是觉得天朝最好，美国快完了，特朗普太坏了干不下去了，欧洲已成地狱，咱中国马上就要收复台湾。 我每次碰到这种话题，就心里偷笑，觉得这些人不到这个政权崩溃的那一天是永远不会醒的，或许他们就应该陪葬？ 然后让我吃惊的，有些高学历有机会出国的，也想着挣了钱早日回来，觉得国内现在待遇越来越好，科研环境也越来越好。我TM直呼书都读到屁眼里去了 先写到这吧，想到什么再说。大家有什么想问的也可以问我 品葱用户 Scandinavian 评论于 2020-11-13 楼主说的见闻限于一线城市吧？ 如果一线城市能直接感受到奔溃那一定是发生战争或者政体剧变之类 品葱用户 吳樂天 评论于 2020-11-14 經濟下滑這個是肯定的 關稅 避險 跟美國對抗導致外企加速外移 這個是長期的趨勢 企業不會馬上撤光 還需要好幾年 現在就看川普能不能有第二個任期 第二任還是川普 ccp再被揍四年 疊加他本身的問題 很容易就洪水滔天 糧食危機要看土共手裡還有沒有美元 有美元就還能買到糧食 2020年

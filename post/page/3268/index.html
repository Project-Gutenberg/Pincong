法国政府据报计划逐渐禁止华为设备参加法国的5G移动网络建设，让华为在2028年全部淘汰出局。 根据路透社的报道，法国网络安全局（ANSSI）已经通知法国大城市的通信运营商说，他们可以使用华为的设备，但使用许可到期之后将不再延续。 知情人士向路透社提供的消息说，法国政府向华为设备发放的使用许可有效期只有三到五年，而欧洲通信公司爱立信和诺基亚的设备使用许可有效期是八年。这等于是法国政府计划逐渐淘汰华为设备，在2028年将华为设备全部排除在法国5G移动网络之外。 路透社说，法国布依格通信（Bouygues Telecom）和法国无线电广播（Altice Europe’s SFR）目前的4G移动网络已经使用了华为设备，而5G网络设备的使用许可又涉及现有移动网络。这就意味着这两家法国通信公司如果放弃华为设备，就必须替换4G移动网络的基础设施。两家公司今年已经几次表示，一旦被迫以高昂的代价替换现有的移动网络，政府就必须提供补偿。 但法国其他通信公司目前基本使用爱立信和诺基亚的通信设备。法国网络安全局本月表示，目前没有使用华为设备的运营商最好避免转用华为设备。一名知情人士说，尚未使用华为设备的运营商很难冒险投资购买华为的设备，因为5G移动新科技至少要八年才能产生投资回报，因此“三年的批准等于是直接的拒绝”。 路透社说，法国网络安全局的决定目前尚未公开宣布。法国总理负责5G设备审批的办公室一名发言人说，法国网络安全局是在法律框架内与电信运营商进行合作，任何目前已经批准的使用许可都不会影响到未来的许可延续或终止决定。 一名知情人士认为，“法国的立场类似英国，但政府的沟通并不相同”，“华为对此将无能为力”。 菊花死期又被往前拉了一下，开心😄👏，接下来看德国的了👌，菊花到今天的这个结局完全是咎由自取，死不足惜😡👎。中共有本事再停播下法甲，要求艾菲尔铁塔灯光关闭👇👇👇。 品葱用户 北美carl 评论于 2020-07-23 来源…… 品葱用户 AT3之王 评论于 2020-07-24 就剩共青团员默老太太了 品葱用户 **Audi2020 北美carl** 评论于 2020-07-23 [ 来源…… ]( “/article/item_id-447485#")5小时前Reuters独家新闻，南华早报，CGTV已转发。 品葱用户 **Audi2020 北美carl** 评论于 2020-07-24 [ 来源…… ]( “/article/item_id-447485#")https://www.reuters.com/article/us-france-huawei-5g-security-exclusive-idUSKCN24N26R 品葱用户 **Audi2020 AT3之王** 评论于 2020-07-24 [ 就剩共青团员默老太太了 ]( “/article/item_id-447491#”) 看她撑多久。 品葱用户 d444c 评论于 2020-07-23 想起前幾天微博粉紅還在那邊說法國跟英美怎麼怎麼不同就覺得好笑

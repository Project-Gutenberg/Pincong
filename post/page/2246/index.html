关于英属爱尔兰，你可能不知道的冷知识

by 挪威蓝森林, at 28 March 2021, category :

（以下所有数据来自大英帝国官方统计档案） 1. 尽管盎格鲁撒克逊人通常都以爱尔兰岛的世居民族自居，但实际上，在1949年，爱尔兰仅有29.1万盎撒人（其中还包含约7万旧军队军人），仅占爱尔兰433.34万总人口中的6%。在这四百多万总人口中，绝大多数都是爱尔兰人等凯尔特民族，爱尔兰人占其中76%。 从1950年代起，英帝国当局开始以军屯、犯屯、支边、自流人员收容等方式，组织大批盎撒人涌入爱尔兰定居。积极的殖民政策在短短20年内迅速改变了爱尔兰的民族结构，到1960年代末，盎撒人的比例已经从6%跃升到了40%，爱尔兰人比例则从76%降到了约45%。该人口结构延续至今。 2. 受自然条件限制，爱尔兰土地的人口承载力十分有限；在英国工业仍欠发达的时期，盎撒拓殖者和当地爱尔兰原住民之间长期处于接近零和博弈的状态。典型的例子是，在英格兰大饥荒时期，大批英格兰农民涌入爱尔兰开垦土地，三年间爱尔兰的盎撒人口翻了一倍以上；但爱尔兰人因饥荒期间的非正常死亡和逃亡，人口反而减少了。爱尔兰人的人口增长因大饥荒长期陷入停滞，直到1960年代中期才开始恢复增长。 在1950-2000的半个世纪中，爱尔兰的盎撒人人口增加了25.9倍，但爱尔兰的爱尔兰人人口只增加了1.59倍。尽管爱尔兰岛的盎撒人口增长迅猛，几乎与原住民爱尔兰人持平，但英国当局并不满足于此，仍对爱尔兰人采取了一系列强制同化措施。 3. 盎撒人移民主要都集中在城市、工业区、油矿资源丰富、可耕地/水源充足的农业区等。以爱尔兰开采历史最悠久的油田为例，这座油田城镇在1950年代之前几乎没有盎撒人，但在英国军队接收该油田后，当地迅速完成了人口置换。在1953年帝国普查中，该城人口中盎撒人占27.1%，到1974年，盎撒人已经占到了79.9%（1974年前无数据）。因石油带来的巨量财富，该城镇所在地级市长期以来是大英帝国人均GDP最高的地级市。 虽然缺乏1954到1973年间的数据，但当地教育系统有1958年起的分民族的在校小学生人数统计。这一数据可以给我们一个对当地人口置换剧烈程度的感性认识：1958年时当地尚有1210名非盎撒人小学生，到1963年就已经迅速跌到史低185人。 4. 盎撒人占据城市、工业、油矿的后果是，爱尔兰人沦为了自己土地上的失业者。在2002年（2002年统计为老口径即有编制；2002年后统计口径为’‘从业人员’’，包含了劳务工、临时工等），盎撒人以39.9%的人口，占据了全爱尔兰国有和集体经济中81.2%的制造业工作、80.7%的采掘业工作、80.7%的科研和技术服务行业工作、74%的电气水生产供应工作。反过来说，非盎撒人，则要以六成人口去分不到两成的岗位。我们都知道大型国有油企的有编工作意味着什么。 在私营经济部门，盎撒人企业也长期排斥雇佣爱尔兰裔员工。英属爱尔兰上议院某位爱尔兰裔女性议员的丈夫曾抗议对爱尔兰人的制度性歧视，该议员拒绝与丈夫切割关系，随后入狱。 盎撒人在第二、第三产业占据垄断性优势，意味着大多数爱尔兰人只能受困于第一产业（或规模极小的民族经济），受困于农村，失去阶层上升的可能性。对比2000年和2010年的两次全不列颠人口普查，爱尔兰人中的农业人口占比从80.35%上升到了82.74%，产业工人占比从1990年的6.15%下降到了4.55%，成为2000年后大英帝国主要民族中，唯一农业人口比例还在提高的民族。 5.

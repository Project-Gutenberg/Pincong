美团等公司的退出计划，或许也正在考量中，2018年腾讯仍然没有理解“新时代”的真正涵义，2021年过后终于懂了。

2016年8月22日中国深圳，一名行人走过腾讯控股有限公司的总部。 摄：Qilai Shen/Bloomberg via Getty Images

一场突如其来的分道扬镳发生在2021年年底。没有任何征兆，腾讯在工作日开盘之前发布了一个减持公告，腾讯与京东长达7年的联姻走到了尽头。

12月23日上午，腾讯控股在港交所发布公告，将以实物分派京东集团股份的方式派发中期股息。按照21股腾讯股份获派1股京东股份比例，腾讯将所持有约4.6亿股京东股权发放给股东。此次派发股票相当于腾讯持有京东集团A类普通股约86.4%。本次派息后，腾讯对京东持股比例将由17%降至2.3%，不再为第一大股东，同时腾讯总裁刘炽平也将卸任京东董事。

按京东12月22日港股收盘价279.2港元/股计，预计腾讯此次中期派息规模约达1284.32亿港元。有人感叹，腾讯分红就分了一个bilibili（此次派息规模约等于bilibili市值）。

第一大股东撤退之后，京东创始人刘强东成为京东第一大持股股东。原本，腾讯通过黄河投资有限公司持股16.9%，拥有4.6%的投票权，为第一大股东；京东集团董事长刘强东持股13.9%，拥有76.9%的投票权；刘强东的持股和投票权都没有变动，但自动成为了京东第一大股东，这令刘强东本来就牢固的控制权又加上了一道保险。京东前几大持股股东变成了刘强东、沃尔玛、Naspers、腾讯，各自持股13.9%、9.3%、4.2%、2.3%。 近两年在监管加码之下，许多互联网科技公司创始人纷纷退位“禅让”，包括阿里巴巴原董事长马云、拼多多原董事长黄峥、字节跳动原董事长张一鸣等，但刘强东仍然牢牢把持京东董事长的位置，重大战略仍要一手掌控。 不过京东也因此丧失了与腾讯的战略合作地位，两者之间原本亲密无间的战友关系，也随之消失。未来京东与腾讯微信在流量入口上的合作，有可能也因此而消失。

究竟腾讯为何会大幅减持京东？在过去十多年的发展中，它是如何走到今天这一步？而在当下中国政府明显加强“监管”的大背景下，这一举措又预示着包括腾讯在内的中国互联网巨头怎样的未来？

2021年8月3日，腾讯游戏徽标出现在手机游戏上。摄：Florence Lo/Reuters/达志影像

2. “巨头”的形成路径

2014年3月10日，在京东上市前夜，腾讯突击入股了京东，占股15%。成为腾讯投资主导，把半条命交给合作伙伴模式的最典型案例。腾讯把旗下的易迅网、拍拍网剥离给京东，完全卸掉电商负担，在微信九宫格给予京东入口。京东相对于阿里巴巴的流量和媒体入口上的劣势马上得到逆转，随着微信用户从2014年迅猛增长，京东的用户和GMV（商品交易总额）随之增长。

京东模式跑通之后，腾讯流量入口+金钱投入模式开始横扫中国互联网大半壁江山，小巨头和创业公司无不望风而倒。美团、拼多多、快手和B站，都拜在腾讯门下，成为腾讯忠实的盟友。

腾讯持有10.54亿股美团B类股份，持股比例为19.74%，为美团的主要股东。刘炽平任美团非执行董事。腾讯亦为快手科技的第一大股东，持股比例为21.81%。腾讯是拼多多第二大股东，持有15.6%的股份。截至2021年1月31日，腾讯持有哔哩哔哩12.4%股权，为公司第二大股东。腾讯持有贝壳找房11.6%的股份，为公司第二大股东。

腾讯的减持举动，引发了资本市场的连锁反应。减持公告发布当天，“腾讯系”股票纷纷下挫，拼多多跌11%，哔哩哔哩跌近9%，京东跌超6%。

这预示着腾讯的投资、经营思路发生了重大转变，2010年3Q大战（腾讯与奇虎360的冲突）之后的投资策略彻底颠覆。

当时，腾讯成为了行业“公敌”，QQ在PC时代成为杀伐四方的利器，腾讯延伸出QQ+新业务的模式，几乎要把整个互联网业务重做一遍。倒在腾讯铁蹄之下的公司数不胜数，腾讯在QQ引入棋牌游戏，早期棋牌游戏老大联众随即熄火；QQ引入邮箱业务，网易邮箱马上受到严重冲击。

2010年7月，信息产业部下属的一家媒体《计算机世界》甚至在头版标题直斥“‘狗日的’腾讯”。财经作家吴晓波所著的《腾讯传》中写道，《计算机世界》的文章发出后，在紧急召开的总裁办公会上，众决策人面前都摆着一封文章的复印件，在长达一刻钟的时间里，没有人发言。最后马化腾喃喃自语：“他们怎么可以骂人？”

而就在那个时候，战无不胜的腾讯，迎来了最顽强最难啃的敌人，奇虎360的周鸿祎。周鸿祎瞅准了舆论和大众对腾讯的不满，果断出击，碰瓷式在360杀毒软件中推出“扣扣保镖”，声称可以阻止QQ扫描用户电脑文件，保护用户隐私。腾讯随即还以颜色，宣布在装有360软件的电脑上将不能运行QQ软件，3Q大战爆发。

尽管腾讯于情于理都占尽优势，事实上，两家公司对簿公堂，最后法院也判决腾讯胜诉。但在实际过程中，腾讯却处于舆论下风，360挟持舆论，利用腾讯的口碑劣迹，加上创始人周鸿祎是语言大师、群众运动大师，在他煽风点火之下，360在媒体和公众舆论中看上去打赢了这场口舌之战。腾讯公司总裁刘炽平后来也表示，“当时的做法现在看来是不周全的，我们缺乏与用户的沟通，在处理重大危机事件的经验上欠缺，也给我们上了非常深刻的一课。”

3Q大战打醒了腾讯，马化腾认为，即便腾讯在3Q大战中属于过错较少的一方，但腾讯四面出击，八面树敌的做法，已经走到了尽头。

腾讯自此开启了第一次转型。

2019年3月21日香港，腾讯控股有限公司董事长兼首席执行官马化腾参与新闻发布会。摄：Justin Chin/Bloomberg via Getty Images

2010年11月11日，马化腾发布了一封致全体员工的信，说：“过去，我们总在思考什么是对的。但是现在，我们要更多想一想什么是能被认同的。”

在邮件最后，马化腾承诺“开放”。但什么是“开放”？2011年初，在一次讨论“什么是腾讯开放能力”的总办会上，马化腾让与会的16名高管，每个人在纸上写下他们认为的腾讯核心能力，总共得出了21个答案。最后他们定下了两个核心能力：一个叫作资本，一个叫作流量。

腾讯的社交软件是流量深海，掌控着中国最大的流量分配；腾讯手中拥有巨量现金，通过投资结盟，为渊驱鱼，最后造成“水大鱼大”。腾讯的“开放”计划，迎来了天时地利人和的阶段。2011年1月21日，腾讯旗下最天才的程序员张小龙主导发布了一款移动社交工具“微信”，这个摇滚音乐爱好者与极简主义产品经理结合体的十年磨一剑，让腾讯拿到了一张珍贵的移动互联网“站台票”。

上线433天，微信用户破亿，2015年6月，微信用户破6亿。微信不但保住了腾讯在PC时代社交媒体的地位，并且进一步发扬光大，腾讯麾下微信与QQ的配置独步中国，在全球仅有Facebook可比肩。

2019年11月22日中国北京，腾讯控股有限公司总裁刘炽平出席彭博新经济论坛的讨论。摄：Takaaki Iwabu/Bloomberg via Getty Images

而在“资本”一端，腾讯总裁刘炽平是掌控“投资”核心能力的人。刘炽平出身美国著名投行高盛，2005年2月加盟腾讯公司，被任命为腾讯首席战略投资官。2006年2月15日出任腾讯公司总裁。刘炽平早年就读于香港英皇书院，是1990年香港中学会考的“九优状元”，后取得美国密歇根大学电子工程学士学位，斯坦福大学电子工程硕士学位以及西北大学凯洛格管理学院工商管理硕士学位。1998年，著名“救火队长”王岐山空降广东担任副省长，处理广东国际信托、粤海集团等不良债务。高盛时任CEO亨利·保尔森指定高盛亚洲从香港派出一个团队协助帮忙，刘炽平就是小组成员之一。

决定“开放”，“把半条命交给合作伙伴”，其实就是打造一套以腾讯流量为中心的生态体系，腾讯挥动着金钱大棒，辅之以流量胡萝卜。

刘炽平在高盛的前同事James Mitchell随着腾讯这次转型加入公司，Mitchell于2011年5月加入腾讯出任CSO（首席战略官），听取所有战略相关的汇报并参与决策，这也是马化腾最初邀请刘炽平加入腾讯时的职位。

这次改革之后，腾讯创始人“五虎上将”基本都退出日常管理，腾讯建立起一套以马化腾、刘炽平、James Mitchell为中心的总办管理体制。马化腾主要做GR和PR工作，日常运作就是刘炽平加Mitchell组合，刘炽平全面负责管理公司日常运营，Mitchell负责包括腾讯的战略规划、战略实施及投资者关系等方面的工作。

三巨头模式建立之后，腾讯开始了史无前例的疯狂扩张，金钱+流量模式取得压倒性胜利，投资与收购的帝国版图跨越中国，触角直达欧洲东南亚和美国。腾讯公司的市值也涨了十倍，在2018年初就突破五千亿美元，曾一度超过Facebook。

2014年5月22日，美国纽约纳斯达克市场，京东创始人兼董事长刘强东在IPO的仪式上。摄：Michael Nagle/Bloomberg via Getty Images

3. 五年盛宴

2012年中共十八大之后，新上任的总理李克强雄心勃勃，喊出“改革是中国最大的红利”，承诺简政放权，减少审批；推广“万众创业，大众创新”。业已成熟的互联网行业迅速进入了“烈火烹油、鲜花着锦”的时代。

一方面许多互联网新业态瞬间爆发，在监管缺失的情况下开出恶之花，在金融自由化大旗之下，P2P放任自流，上万家P2P公司榨取了无数的家庭几十年心血。另一方面，已经成为巨头的公司趁机大肆扩张，不管是阿里巴巴的买买买，还是腾讯的投投投，在短短几年内就走到极度垄断状态。

2013年开始，至2018年终结的中国放任自由互联网阶段，腾讯是最大的受益者。2015年中国互联网市场有三场轰轰烈烈的大合并。滴滴合并快的，美团吃掉大众点评，58吞并赶集网。在自由竞争的市场，旗鼓相当、资金充裕的双方，短期内很难分出胜负；但是几大资本的联合下，三场合并都在短短几个月之内扫尾。红杉中国、高瓴资本等风投串联其间，成为三场合并的关键先生。合并后的几大公司，纷纷在微信九宫格拿到宝贵入口，而腾讯几乎都是最大股东。

以腾讯为代表的大企业投资部门，和以红杉中国和高瓴资本为代表的外资风投、以金沙江创投等为代表的本土风投，形成一股合力，推动互联网连横合纵，形成新的格局。

在那些时候，国家发改委、工商总局、商务部三家合管反垄断的机构没有一次提出异议。

2017年，腾讯投资部负责人李朝晖透露腾讯投资金额已过千亿，投出的独角兽规模公司超过50家。刘炽平在2018年初的腾讯投资年会上也曾透露，所投企业所新增的价值已超过腾讯本身的市值。

2017年12月3日，第四届世界互联网大会在乌镇召开。这是2017年10月份中共十九大之后第一件大型会议。当日晚上，京东创始人刘强东和美团创始人王兴做东，邀请参会的互联网企业家、投资人参加饭局。这场互联网历史上著名的“东兴饭局”，是那段互联网辉煌的顶峰。

马化腾端坐中间，刘强东王兴一左一右，参与饭局的还有滴滴程维、快手宿华、58姚劲波、小米雷军、字节张一鸣、联想杨元庆，以及投资人红杉中国沈南鹏、高瓴资本张磊、金沙江创投朱啸虎等人。在饭局中，除了小米、联想和字节跳动之外，大部分都是腾讯投资的企业。

东兴饭局是中国互联网竞争发展的巅峰时刻，不管是红墙之内还是江湖之远，互联网企业家们都赢得了喝彩和掌声：互联网企业家多次参加总理办公会，党媒对“大众创新、万众创业”的互联网企业不吝溢美之词，民众对履历光鲜没有原罪的互联网企业家同样热烈追捧。

4. 阿里暗线

与腾讯不同，阿里巴巴向来强调“终局思考”，马云又是一贯游走在官商之间最游刃有余的“聪明人”，腾讯的结盟是这五年的“明线”，阿里巴巴就是这五年的“暗线”。阿里巴巴的投资强调与业务协同，财务指标不是考核标准，跟腾讯比起来，阿里巴巴许多投资标的溢价不多，甚至许多是亏损的，比如投资苏宁易购，几乎亏掉一半。

在投资上比腾讯稍逊一筹，实际上掩盖了阿里巴巴另一种隐秘的结盟方式。

在这五年里，阿里巴巴是与权贵资本结盟，试图掌控话语权和解释权，如果没有后来的形势剧变，阿里巴巴的这种结盟，是一种自觉行为，也包含更大的野心；当然，当形势急转直下，阿里也遭受到更大的压力，触碰“龙鳞”，染指“国之重器”，令阿里的局势至今晦暗不明。

2014年，阿里巴巴上市之前，引入了三家中国著名投资基金，分别是国家开发银行下属的国开金融，中信集团的中信资本，以及博裕资本，前两者是中国最著名的红色金融资本，而博裕资本合伙人之一江志成则是江泽民之孙。

马云终于得到了心心念念的国家队支持，与官僚资本和权贵资本觥筹交错，接近权力圈带来了眩晕感，马云忘记了自己的信条之一，就是“要跟政府谈恋爱，但不要结婚”。

2015年，尝到甜头的马云更进一步，蚂蚁金服A轮融资，更是引入了国家资本，本轮融资一共引入了8家机构，其中全国社保基金持股比例最高，为5%；国开金融与四家保险公司（中国人寿、中国人保、中国太保与新华人寿）各投资约9亿元，每家约占0.5%；此外还包括春华基金和国资背景的上海金浦产业基金等两家PE机构。

2016年蚂蚁金服B轮融资，新加入的战略投资者包括中投海外和建设银行下属建信信托，当时中投持股仅次于社保基金，该轮之后持股3%。

这几轮融资是全国社保基金第一次以直接投资的方式投资一家民营企业；中投是中国国家主权基金，目标也是海外资产，如此大规模国内投资，也是首次。

其后蚂蚁多次冲撞监管，创新横溢斜出，打法五花八门，马云明白自家的保险垫足够厚实。

社保基金、中投超豪华国家队的投资让蚂蚁有了政治“保险锁”。2020年，就在马云演讲“翻车”的那场外滩金融峰会，全国社保基金副理事长陈文辉仍在与蚂蚁唱和，希望蚂蚁上市多给配额，甚至喊出“给社保基金就是给全国人民”这种话来。

除了与国家资本联姻，阿里还出手参股收购多家媒体，试图掌控话语权。阿里巴巴先后入股了新浪微博、第一财经、芒果传媒、分众传媒，全资收购《南华早报》，全面介入新媒体虎嗅、钛媒体、品玩网等。试图稳住国家“钱袋子”的同时，掌控住市场的“笔杆子”。

2021年10月12日，位于中国深圳的腾讯控股有限公司总部。摄：Qilai Shen/Bloomberg via Getty Images

5. 盛筵的尾声

2013年至2018年的腾讯和阿里审时度势，一个交好社会资本和外国资本圈，腾讯系公司红旗插遍神州；一个走上层路线，讨好国家资本和权贵资本，把马云的话语送到普罗大众面前。

2018年中共的十九届三中全会召开，全会审议通过了《中共中央关于深化党和国家机构改革的决定》和《深化党和国家机构改革方案》，市场监管总局扩权，从国家发改委、商务部手中集中了反垄断权力，成为名副其实的国家反垄断局，为2021年的巨额反垄断处罚埋下伏笔。

中共中央宣传部统一管理新闻出版工作，将国家新闻出版广电总局的新闻出版管理职责划入中央宣传部，中央宣传部对外加挂国家新闻出版署（国家版权局）牌子。

原本属于国务院序列的新闻出版总署变为中宣部直接管理机构，其管理的游戏版号随之冻结；中央媒体开始转向，人民网三批《王者荣耀》，腾讯旗舰游戏产品《绝地求生》迟迟拿不到版号变现，在一年版号冻结期腾讯股价跌到冰点，2018年末市值缩水1万亿港元。

有自媒体人提出《腾讯没有梦想》，试图从战略和管理角度，剖析腾讯跌落的原因。

2018年9月30日腾讯开启了第二次改革计划，重新调整架构，提出“科技向善”，保留原有的企业发展事业群（CDG）、互动娱乐事业群（IEG）、技术工程事业群（TEG）、微信事业群（WXG）；新成立云与智慧产业事业群（CSIG）、平台与内容事业群（PCG）。

腾讯的930变革，是感受到了意识形态环境变化、垂直领域监管收紧而寻求的变阵，但腾讯真正的流量+投资的战略并没有变化，刘炽平仍然不断在寻求新的标的，拓展腾讯投资边界。

腾讯音乐仍然按计划分拆上市，腾讯整合游戏直播资源，分别成为虎牙和斗鱼的第一大股东，推进二者合并事项。

但实际上，第二次改革之前的腾讯也没有变过，变化的是大背景。

直到2020年11月3日晚，蚂蚁上市被叫停，针对互联网巨头接下来的政策才真正显露出来。

2020年12月的中央经济工作会议，将“强化反垄断与防止资本无序扩张”作为重点任务。

2021年，阿里巴巴和美团因为“二选一被”处罚，其中阿里巴巴缴纳182亿元罚款创天价记录；

虎牙和斗鱼进行中的合并泡汤，腾讯整合游戏直播的努力白费；腾讯音乐的独家版权被限令三个月解除，腾讯音乐股价跌去三分之二；游戏版号再一次停发，已经达到半年之久，眼看直追2018年；戏监管严厉程度已经达到极限，未成年人游戏时间被压缩到每周三个小时。

腾讯市值缩水四成，阿里巴巴股价更是跌去超过一半。

2017年10月1日中国天津，年轻人在商场外练习，这里举行手机游戏的比赛。 摄：Zhang Peng/LightRocket via Getty Images

6. 第三次“改革”？

经过2021年，腾讯也意识到必须对2011年形成的流量+投资模式做出彻底的改变，才能真正适应当前的监管体制和上层意志。

2021年，腾讯率先推出“共同富裕”基金，承诺投入1000亿元启动“共同富裕专项计划”，用于乡村振兴、低收入人群增收、基层医疗体系完善、教育均衡发展等民生领域。腾讯这一举动得到阿里巴巴和拼多多的回应，阿里也宣布投入1000亿元成立“共同富裕”基金。

在成立共同富裕基金的时候，腾讯或许已经下定决心，这一次要往“对”的方向进行改变，而京东绝对不会是唯一一家它会减持并降低自身影响力的公司。

今年1月4日，腾讯宣布将减持新加坡冬海集团（Sea Limited）部分股票，自身在冬海的投票权也大幅降低。冬海有“东南亚小腾讯”之称，旗下拥有电商、游戏等业务，是目前东南亚市值最高的互联网公司。腾讯于2010年入股冬海，持有21.3%的股份，并享有“一股多票”的超级投票权，此前投资12年间从未减持。

冬海集团位于新加坡，其业务也不在中国开展，按理说跟中国关系不大，腾讯仍然大刀砍去。可以肯定的是，冬海不会是最后一家。

美团、快手、拼多多、bilibili的退出计划，或许也都在酝酿或考量之中，2018年腾讯仍然没有理解“新时代”的真正涵义，2021年过后终于懂了。

在2021年底的员工大会上，马化腾说，腾讯只是国家社会大发展期间的一家普通公司，是国家发展浪潮下的受益者，并不是什么基础服务，随时都可以被替换。他还说，未来，腾讯在服务国家和社会的时候，要做到不缺位、做到位、不越位，做好助手，做好连接器。

1985年，14岁的马化腾缠着父母买了一台准专业级天文望远镜，花了马爸爸700多元，相当于4个多月工资，这让马化腾在1986年成为全校第一个看到哈雷彗星的人。

2022年，刚过50岁的马化腾恐怕用不着将望远镜对准未来，就已经能感受到时代的寒流将至。腾讯快步走进了新时代。

（超先声，互联网科技观察者）

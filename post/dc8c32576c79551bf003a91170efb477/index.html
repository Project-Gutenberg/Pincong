NO.1534-重庆发生了什么

作者：土圭垚

 制图：孙绿 / 校稿：猫斯图 / 编辑：养乐多

西南大都会重庆，在各种意义上都是一座令人印象深刻的城市。

别的不用多说，就只说这座城市明显的山地属性，已经为它带来了3D魔幻森林的网红名号。更不用提洪崖洞、磁器口、朝天门这些听名字就很雄壮的经典重庆符号，个个都是闻名遐迩的。

一座立体的城市

（图片：Stepanov Aleksei / Shutterstock）▼

但最近，重庆又因为一件让重庆人自己不太高兴的事出了名。

由于连降暴雨，重庆綦江段水位暴涨，形成了局部洪水，对当地群众生产生活造成了重大影响。而在重庆发生的一切，很可能只是困难的2020的一个缩影。

重庆为什么会下暴雨？

人们对重庆最经典的印象，当然非重庆的主城区莫属，也就是嘉陵江汇入长江的江段。

重庆的核心区是嘉陵江与长江的交汇点

但作为直辖市的重庆远比这要大

（图片：asiastock / Shutterstock.）▼

事实上，重庆这个概念的覆盖面积比人们印象中的重庆大很多。表面上看上去只是一座城市的重庆，其实有着不亚于一个小型省份的辖境，内部的地区区分就更多了。至少在重庆人自己心中，各地区也有着明显的差别，而且各自有着自己的中心。

有些地方虽然很远，但也设为“区”

相对平缓地带的区更多，也更密集▼

但无论是中心城区的魔幻森林还是大重庆概念中的外围区县，山始终是它的核心元素。

习惯了就好了

（图片：LP2 Studio / Shutterstock.）▼

重庆地势由南北向长江河谷逐渐降低，西北部和中部以丘陵、低山为主，东南部靠大巴山和武陵山两座大山脉。总的地貌特征已被当地人归结为顺口溜：山丘广布，类型复杂，地形崎岖，高差悬殊，河流纵横，切割强烈。至于平原，也就是西南方言中的“坝（子）”，仅占全市土地面积的2.39%，可谓是“地无三尺平”。

西北部和中部还稍显平缓

东部和东南部就真的是崇山峻岭了▼

地形变幻莫测的山地，自然景区众多，也利于生态环境涵养。但在山区，地形对降水的影响也是相当显著的，海拔、坡向以及地表性质等对降水都有影响。

一般而言，降雨量常常随海拔的上升而增加，这种地形的增雨效应在重庆的几大山区中，如：大巴、巫山、武陵、大娄都非常显著。而由于山地的增雨作用，山区的降雨量又远大于丘陵区和平原区，若是暴雨降临在河流上游（在山地这种现象还很常见），更容易形成洪水，并向河流下游传播，造成严重的洪水灾害。

这一状况在西南地区颇为常见

另一个典型例子就是重庆旁边的四川

巨大的海拔使降水集中在成都平原及以南地区▼

此外，受到亚热带湿润季风类型气候影响，重庆的暴雨时间分布颇有规律，最早出现在3月，最晚结束于11月，主要集中在6~9月。

5月份暴雨多发区有三个，第一个在北碚、璧山一带，第二个在长寿，第三个在万州。随着雨量加大，6月份的时候，5月形成的三个暴雨多发区就会连成一片，构成一个暴雨带，即铜梁、北碚、渝北、长寿、忠县、石柱、万州一带。与此同时，另一个暴雨区开始出现，即黔江、酉阳、彭水、武隆一带。

西北一片，东南一片▼

7月份的时候，各地进入降雨高峰期，暴雨次数、雨量、强度、分布范围都显著增加，并形成两个暴雨中心。第一个是渝东北的垫江、忠县、万州、梁平、开州一带，另一个是渝西的潼南、北碚、巴南一带。

东北一块，西边一块▼

不过到了8月份的时候，随着副热带高气压进一步西伸北抬，雨带向西推移，暴雨光顾的地区主要在渝西，仅在合川、璧山、永川、荣昌一带形成一个多暴雨的地区；而前几个月受暴雨侵扰的渝东部和中部地区则在副高的控制下，进入伏旱季节，暴雨就相对少很多了。

仿佛是最热的时候…▼

到了9月份，雨带会东缩南移，渝东北地区和渝东南地区又会进入一个多暴雨的时段，出现一个小高峰。

总体而言，重庆市暴雨的年内分布结构有区域差异，渝东北和渝东南年内分布多为双峰型，主峰在6月或7月，次主峰在9月；渝中和渝西多为单峰型，主峰在7月。

为什么是綦[qí]江？

綦江，既是一个地理名词，又是一个行政区名词。

作为地理名词的綦江，是长江右岸的一级支流，发源于重庆市綦江区万隆村大垭口。但它之后并没有一直在重庆境内游荡，而是进入贵州省的习水县和桐梓县，再由綦江区的安稳镇重新回到重庆，并在江津区的支坪汇入长江。

作为行政区名词的綦江，最早出现于元朝末年，而且与作为河流的綦江有着千丝万缕的联系。这个行政单位后来逐渐演变为綦江县，并长期以来就归重庆（东川）所管辖，后来随着经济发展，与万盛区合并为新的綦江区，驻地依然还在老綦江。

綦江与綦江区

其实离重庆很近的▼

綦江与朝天门的直线距离足足有60多公里，明显是重庆的远郊地区，还是它的老工业基地及矿产资源聚集区。

所以最近这一次綦江洪灾，虽然没有威胁到重庆主城区，但破坏的地区也不是一些偏僻的乡镇或是人口较少的地区，而是有着常住居民众多的綦江主城区，灾害十分严重。

綦江沿岸的綦江区城区

（图片：google map）▼

今年进入雨季之后，6月20日以来贵州、重庆至长江中下游一带就出现了较强降雨，这与綦江流域有部分重叠。

到了6月21日，渝东南和渝西出现中到大雨，局部出现暴雨到大暴雨。酉阳、秀山、彭水等9个区县有暴雨，而其中的酉阳、秀山、武隆、南川4个区县属于大暴雨。

在当日通惠河、清溪河、藻渡河、南川等多条属于綦江水系的中小河流，就出现了超警戒水位洪水的情况。

6月21日重庆市地质灾害气象风险预警图

21日20时-22日20时预报

（制图参考：重庆市规划和自然资源局）▼

大暴雨的降临，再加上各条支流的水量不断汇流，使得主流的水量即不断增加。到了6月22日中午时段，重庆市水文监测总站将江津区綦江五岔站洪水橙色预警升级为綦江流域重庆段全线洪水红色预警。

果然，当天下午4时左右，洪峰经过綦江城区，最高水位超过保证水位近5米。洪峰还在继续，过了綦江城区后，当日晚上8点，江津区綦江五岔站出现洪峰水位205.85米，超过保证水位（200.51米）5.34米。这个时候它还破了一个记录，超过1998年那次洪水水位0.30米，已达建站以来最高水位。

6月22日重庆市地质灾害气象风险预警图

22日20时-23日20时预报

（制图参考：重庆市规划和自然资源局）▼

后果很严重

重庆綦江告急。

首先是交通方面，綦江的运煤铁路专线赶松铁路支线（赶水-张家口）因这次降雨影响被泥石流冲断。所幸它只是以运煤业务为主的支线，对主线的客货运交通并无太大影响，也没有造成客运列车倾覆和人员伤亡。公路方面，这一地区也遭受重创，多条主干道被毁。

由于洪峰过境綦江城区，这次洪灾还对城市建成区造成了严重的破坏。街道被淹，城区众多楼房被洪水漫过，有的甚至仅剩了楼顶，一楼的商铺损失尤其惨重。

幸运的是洪峰顺利过境，并未在綦江城区长时间停留。而这次洪水也顺利汇入长江，未造成进一步的连续灾害。

綦江汇入长江处

（图片：google map）▼

由于綦江历史上多次发生过洪灾，当地对预防洪灾这类基础工作极为重视，没有半点马虎，上文提到的五岔水文站也在这次洪灾的预警中立了大功。预算到綦江市民将会与这次洪灾可能发生遭遇战，重庆全市及綦江全区应急、水文、气象、抢险等部门在洪水进城之前，紧急转移10万余人，疏散离开市区，无一人伤亡。

在6月23日，洪峰过境之后，进入安全状态的綦江，相关部门立刻展开清淤消杀和排险工作，群众的生产自救工作也随之展开，市民的生活也将逐步恢复正轨。

6月24日重庆市地质灾害气象风险预警图

24日20时-25日20时预报

（制图参考：重庆市规划和自然资源局）▼

然而，霏霏淫雨仍然没有迎来终结时，同属綦江区的万盛经开区近期也因暴雨出现了地质灾害。

同在西南地区的贵州沿河和桐梓也发生大规模内涝灾害，湖南近期也首发暴雨橙色预警。

面对多处并行的挑战，国家水利部也出面发声，表示今年我国水文年景总体偏差，防汛形势严峻，需要在各个行政级别做好充分的应急准备，严防特大洪涝灾害，保护人民生命财产安全。
